# SpyGlass Pharma (SGP)

## 카탈리스트
SGP-005/SGP-006 Phase 3 등록 진행 중 (주요 데이터 2028~2029년 예상); BIM-DRS Phase 1 개시 예정 (2026)

## 회사 개요
SpyGlass Pharma는 백내장 수술 시 동시 삽입하는 BIM-IOL(Bimatoprost Intraocular Insert) 시스템으로 녹내장·안압 치료 혁신을 목표로 하는 안과 기기 기업. 2026년 2월 IPO($172.5M 조달). Phase 1/2에서 12개월 양성 결과. 두 개의 Phase 3 시험(SGP-005, SGP-006) 등록 개시.

## 매출
수익 없음. IPO 자금으로 Phase 3 진행.

## 플랫폼
- BIM-IOL: 안내렌즈 일체형 약물 지속방출 시스템 (bimatoprost 최대 3년 방출)
- BIM-DRS: 백내장 수술 없는 독립형 안구내 지속방출 기기

## 적응증
Ophthalmology (개방각 녹내장, 안압 상승)

## 파트너
없음 (독자 개발·임상)

## 상업화 제품

## 임상 디자인
- SGP-005/006 Phase 3: 무작위, 이중맹검, 위약 대조; ~400명/연구, 45개 미국 사이트, 36개월 추적
- 1차 평가변수: 안압 변화 (IOP reduction)
- Phase 1/2 12개월 결과: Positive (2026.3)

## 타겟 질환
**개방각 녹내장 (Open-Angle Glaucoma, OAG) / 안압 상승 (OHT)**

안압 만성 상승 → 시신경 손상 → 실명.

- 환자 수: 미국 약 300만 명(녹내장) + OHT 다수
- 병리: 방수 배출 저항 증가 → 안압 상승 → 시신경 허혈

## 기존 치료제
점안액(prostaglandin analogues, β차단제), 레이저, 수술(trabeculectomy). BIM-IOL은 백내장 수술 시 삽입해 3년 약물 자동 투여로 순응도 문제 해결.

## 사전 공개 임상
- **BIM-IOL Phase 1/2 · 12개월 · 양성 (2026.3)**
  결과: Positive - 안압 유의미 감소, 안전성 우호적.
- **SGP-005/006 Phase 3 · 2026.1 등록 개시**
  36개월 추적. 주요 데이터 2028~2029년 예상.

## Modality
**BIM-IOL System**
Sustained-release bimatoprost intraocular device integrated with IOL (type: Other)

안내렌즈 후면에 bimatoprost 저장소 내장. 백내장 수술 시 동시 삽입 → 최대 3년 지속 약물 방출.

## MOA
- 타겟: 방수 배출 경로 (trabecular meshwork + uveoscleral)
- 기전: Bimatoprost(프로스타글란딘 유사체)의 FP 수용체 작용 → 방수 유출 촉진 → 안압 하강. 지속방출 기기로 매일 점안 없이 3년 효과 유지.
- 병태생리: OAG에서 방수 배출 저하 → 안압 상승 → 시신경 손상. 점안액 순응도 낮아 치료 실패 빈번.
