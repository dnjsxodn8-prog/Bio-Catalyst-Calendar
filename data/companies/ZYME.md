## 회사 개요

Zymeworks(나스닥: ZYME)는 캐나다 밴쿠버에 본사를 둔 임상단계 바이오텍. HER2 이중특이 항체 zanidatamab(Ziihera)으로 FDA 가속승인을 획득한 플랫폼 기반 종양학 전문 기업이다.

**핵심 전략 전환(2025~):** 전통적 바이오텍 모델에서 "로열티 주도 조직(royalty-driven organization)"으로 전환. Ziihera의 글로벌 판매는 Jazz Pharmaceuticals(ex-Asia)와 BeOne Medicines(아시아)가 담당하고, Zymeworks는 로열티와 마일스톤을 수취하며 차세대 파이프라인(ADC, T세포 인게이저)을 개발한다.

- 설립: 2003년
- 상장: 나스닥 ZYME
- 시가총액: ~$2.09B (2026년 4월 기준)
- 주식수: 75.21M주 (2026년 1월 기준)
- 현금: $270.6M (2025년 말) + Royalty Pharma $250M 로열티 담보 파이낸싱(2026년 3월)
- 핵심 플랫폼: Azymetric™(이중특이 항체), ZymeLink™(ADC), EFECT™

## 플랫폼

**1. Azymetric™ (이중특이 항체 플랫폼)**
- IgG1-like 이중특이 항체를 안정적·효율적으로 생산
- zanidatamab(ZW25)이 대표 산물 — HER2 ECD2 + ECD4 동시 결합(biparatopic)
- 단일 IgG 포맷으로 정상 IgG처럼 생산 가능, ADCC/CDC 이펙터 기능 보유

**2. ZymeLink™ (ADC 플랫폼)**
- 토포이소머라아제-I 억제제(Topo-I) 기반 페이로드(ZD06519 등)
- Bystander activity 보유, 부위 특이적 결합(site-specific conjugation)
- ZW191(FRα-ADC), ZW251(GPC3-ADC)에 적용

**3. EFECT™ (Fc engineering)**
- 이펙터 기능 강화/조절을 위한 Fc 변형 기술

## 임상 디자인

### HERIZON-BTC-01 (승인 근거 시험)
- **디자인:** 다기관, 단일군(single-arm), 오픈라벨
- **Arms:** 1개 — zanidatamab 20mg/kg IV Q2W 단독요법
- **N:** 87명 등록 (Cohort 1: IHC 2+ 또는 3+, HER2-양성 80명; Cohort 2: IHC 0·1+ 7명)
- **Phase:** 2b (pivotal)
- **Primary:** 독립 중앙 검토(ICR) 기반 확인된 ORR (Cohort 1)
- **Secondary:** DoR, PFS, OS, 안전성
- **NCT:** NCT04466891
- **등록 기간:** 2020년 9월 ~ 2022년 3월

### HERIZON-GEA-01 (GEA 1L 신청 근거 시험)
- **디자인:** 글로벌 무작위배정, 오픈라벨, 활성대조군, Phase 3 (Zymeworks·BeOne 공동)
- **Arms:**
  - Arm A: Ziihera + tislelizumab + 화학요법 (n=302)
  - Arm B: Ziihera + 화학요법 (n=304)
  - Arm C (대조): trastuzumab + 화학요법 (n=308)
- **N:** 914명 (30개국 이상, ~300개 기관)
- **Phase:** 3
- **Primary:** PFS (ICR), OS (각 비교)
- **Secondary:** ORR, DoR, 안전성
- **NCT:** NCT05152147
- **등록 기간:** 2021년 12월 ~ 2025년 2월

**HERIZON-GEA-01 주요 결과 (데이터 컷오프 2025년 10월, 중앙 추적기간 26개월):**
- 중앙값 PFS: Arm A 12.4개월 / Arm B 12.4개월 vs. 대조군 8.1개월 (양측 모두 통계적 유의)
- 중앙값 OS: Arm A 26.4개월 vs. 대조군 19.2개월 (통계적 유의)
- Arm B OS: 통계적 유의에는 미치지 못했으나 임상적으로 의미 있는 경향
- ORR·DoR: 양 investigational arm 모두 대조군 대비 개선
- 안전성: 개별 약제의 기지(旣知) 프로파일과 일치, 새로운 신호 없음

## 타겟 질환

**1. 담도암 (BTC, Biliary Tract Cancer) — 승인 적응증**
- **환자 수:** 미국 약 8,000명/년 진단. 전 세계적으로 증가 추세
- **병리:** 담관암종(iCCA·eCCA)·담낭암(GBC)·팽대부암 포함 이질적 희귀종양군. HER2 양성 비율: GBC ~55%, eCCA ~17%, iCCA ~5%. 전체 BTC 중 HER2 IHC 3+ 환자 비율 약 5~15%.
- **미충족 수요:** 2L 치료 후 표준요법 부재. Ziihera가 최초 HER2 표적 승인 약제

**2. 위식도 선암종 (GEA, Gastroesophageal Adenocarcinoma) — PDUFA 2026-08-25**
- **환자 수:** 미국 위암 약 26,000명/년. HER2 과발현(IHC 3+ 또는 FISH+) 비율 15~20% → HER2+ 1L GEA 약 5,000~6,000명/년
- **병리:** 위암·위식도 접합부암·식도 선암 포함. HER2 증폭 시 불량한 예후, 적극적 항HER2 요법이 근거중심 치료.
- **기존 표준:** trastuzumab + 화학요법 (ToGA 시험 근거). Ziihera는 이를 대체 목표

**3. 난소암 (Ovarian Cancer, PROC) — ZW191 개발 중**
- **환자 수:** 미국 약 20,000명/년 진단. 백금저항성 난소암(PROC) 전체의 약 25%
- **병리:** FRα 과발현이 특징적(~80% 이상). Mirvetuximab(IMGN853, Elahere) 승인으로 FRα ADC 유효성 입증됨

## 기존 치료제

**BTC 2L 경쟁 (Ziihera 승인 공간):**
- 직접 경쟁: 없음 (Ziihera가 BTC에서 HER2 표적 최초 승인)
- 간접: 펨브롤리주맙(MSI-H/dMMR), FGFR2 억제제(intrahepatic CCA), IDH1 억제제

**GEA 1L 경쟁 (PDUFA 2026-08-25):**
- 현행 표준: trastuzumab + 화학요법 (ToGA, HERIZON-GEA-01 대조군)
- 경쟁자: pembrolizumab + trastuzumab + 화학요법 (Keynote-811 승인), nivolumab (CheckMate-649)
- 차별화: Ziihera의 이중 HER2 결합(biparatopic)으로 trastuzumab 대비 우월한 HER2 하향조절 및 CDC 활성

**난소암 FRα ADC 경쟁 (ZW191):**
- Elahere® (mirvetuximab soravtansine, ImmunoGen/AbbVie): FRα ADC, FRα-high PROC 승인
- ZW191 차별화: Topo-I 억제제 페이로드(vs. DM4), bystander activity, FRα 발현 폭 넓음

## Modality

- **zanidatamab (Ziihera):** HER2 biparatopic bispecific IgG1-like 항체. IgG 포맷(scFv 융합). 분자량 ~150kDa.
- **ZW191:** FRα 표적 ADC. 토포이소머라아제-I 억제제(ZD06519) 페이로드. ZymeLink™ 플랫폼.
- **ZW251:** GPC3 표적 ADC. Topo-I 억제제(ZD06519) 페이로드. ZymeLink™ 플랫폼.
- **ZW209:** DLL3 표적 삼특이적 T세포 인게이저(trispecific TCE). CD3 + CD28 + DLL3 동시 결합. SCLC 대상.

## MOA

**zanidatamab (Ziihera) — HER2 biparatopic bispecific**

| 요소 | 내용 |
|------|------|
| **타겟** | HER2 ECD2(dimerization domain, trastuzumab 결합 도메인) + HER2 ECD4(juxtamembrane, pertuzumab 결합 도메인) — 동일 분자의 두 에피토프 동시 결합 |
| **결합 방식** | 인접 HER2 분자들을 trans-결합 → HER2 재배열 유도(clustering/capping). 세포 표면 항체 포화도 ~1.5배 증가 |
| **기전 1: 신호 차단** | ECD2 결합(trastuzumab 유사) → HER2-HER3 이형이량체화 억제. ECD4 결합(pertuzumab 유사) → HER2 리간드 독립적 신호 차단 |
| **기전 2: 수용체 하향조절** | biparatopic 결합 → 강화된 HER2 내재화(internalization) + 세포 표면 HER2 단백질 감소 |
| **기전 3: 이펙터 기능** | IgG1 Fc → ADCC(항체 의존 세포독성), ADCP(항체 의존 세포식균) |
| **기전 4: CDC** | HER2 클러스터링이 C1q 결합 촉진 → 보체 의존성 세포독성(CDC) 유도. trastuzumab·pertuzumab 병용 대비 우월한 CDC |
| **병태생리** | HER2 과발현(IHC 3+ 또는 증폭) → PI3K/AKT/mTOR·RAS/MAPK 경로 지속 활성화 → 종양 증식·생존. BTC에서 HER2 증폭은 eCCA·GBC에서 특히 빈번 |

**ZW191 — FRα-targeting ADC**
- 타겟: Folate Receptor α (FRα) — 난소암·자궁내막암에 고발현, 정상 조직 제한적
- 기전: 항FRα 항체 결합 → 내재화 → 리소솜에서 ZD06519(Topo-I 억제제) 방출 → DNA 단일가닥 절단 → 세포사멸. Bystander activity로 인접 FRα 저발현 세포도 사멸.
