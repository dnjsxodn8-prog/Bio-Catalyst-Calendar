# Zymeworks (ZYME)

## 카탈리스트
- 2026-08-25 PDUFA — Ziihera (zanidatamab) 1L HER2+ GEA (Jazz sBLA, Priority Review)
- 2026 H2 — ZW191 Phase 1 Part 2a 결과 발표 (용량 최적화, 난소암)
- 2026 H1 — ZW209 (DLL3 삼특이적 T세포 인게이저) IND 신청 예정
- 2026-04-21 — ZW191 AACR 2026 Phase 1 데이터 발표 완료 (난소암 cORR 61%)
- 2025-11-17 — HERIZON-GEA-01 Phase 3 긍정 결과 발표 완료

## 회사 개요
HER2 이중특이 항체 Ziihera(zanidatamab)로 BTC 2L 가속승인을 보유한 종양학 바이오텍으로, Jazz에 상업화를 위임하고 로열티·마일스톤을 수취하는 구조로 전환 중이다. 2026-08-25 PDUFA인 GEA 1L sBLA 승인이 핵심 촉매로, 성공 시 최대 $440M 마일스톤과 함께 Ziihera의 주요 시장 진입이 실현된다.

## 매출
FY2025 $106.0M (+39% YoY), FY2024 $76.3M. 마일스톤·로열티 수취 구조. 순손실 $81.1M (FY2025).

## 플랫폼
- **Azymetric**: IgG1-like 이중특이 항체 플랫폼. zanidatamab(HER2 ECD2+ECD4 동시 결합)이 대표 산물.
- **ZymeLink**: 토포이소머라아제-I 억제제(ZD06519) 기반 ADC 플랫폼. Bystander activity 보유.
- **EFECT**: 이펙터 기능 강화·조절을 위한 Fc 변형 기술.

## 적응증
- BTC 2L (HER2 IHC 3+) — Ziihera FDA 가속승인 (2024.11)
- GEA 1L HER2+ — Ziihera sBLA Priority Review (PDUFA 2026-08-25)
- 난소암 PROC / 자궁내막암 — ZW191 (FRα ADC, Phase 1)
- 간세포암 HCC — ZW251 (GPC3 ADC, Phase 1)
- SCLC — ZW209 (DLL3 삼특이적 TCE, IND 예정)

## 파트너
- **Jazz Pharmaceuticals**: 2021년 체결, $325M 선급금. 아시아(일본 제외) 외 글로벌 권리. Jazz가 sBLA·상업화·개발비 담당.
- **BeOne Medicines (구 BeiGene)**: 아시아(일본 제외), 호주, 뉴질랜드 권리. 중국 조건부 승인 보유.
- **Royalty Pharma**: 2026년 3월 $250M 로열티 담보 파이낸싱. Ziihera 로열티의 30%로 상환(비소구권).

## 임상 디자인
**HERIZON-BTC-01 (BTC 2L, Phase 2b, NCT04466891)**
- 디자인: 다기관, 단일군, 오픈라벨. zanidatamab 20mg/kg IV Q2W 단독요법
- N: 87명 등록 | Primary: ICR 기반 확인된 ORR | 결과: ORR 41.3%, mDoR 14.9개월, mOS 15.5개월 → FDA 가속승인 근거

**HERIZON-GEA-01 (GEA 1L, Phase 3, NCT05152147)**
- 디자인: 글로벌 무작위배정, 오픈라벨, 활성대조군 (Zymeworks·BeOne 공동)
- N: 914명 (Arm A: Ziihera+tisle+chemo / Arm B: Ziihera+chemo / Arm C: tras+chemo 대조) | Primary: PFS, OS | 결과: mPFS 12.4개월 vs 8.1개월, mOS 26.4 vs 19.2개월 (Arm A, 통계 유의)

## 타겟 질환
**담도암 (BTC)**
- 환자 수: 미국 약 8,000명/년 진단. HER2 양성 비율: GBC ~55%, eCCA ~17%.
- 병리: 담관암종·담낭암 포함 이질적 희귀종양군. 2L 치료 후 표준요법 부재. Ziihera가 최초 HER2 표적 승인 약제.

**위식도 선암종 (GEA)**
- 환자 수: 미국 위암 약 26,000명/년. HER2 과발현 비율 15~20% → HER2+ 1L GEA 약 5,000~6,000명/년.
- 병리: HER2 증폭 시 불량한 예후. 기존 표준은 trastuzumab + 화학요법 (ToGA). Ziihera는 이를 대체 목표.

## 기존 치료제
- Ziihera 경쟁 BTC 2L: 직접 경쟁 없음 (최초 HER2 표적 승인)
- GEA 1L 경쟁: trastuzumab + 화학요법 (ToGA, 대조군), pembrolizumab+tras+chemo (Keynote-811)
- 난소암 FRα ADC: Elahere® (mirvetuximab soravtansine, AbbVie) — ZW191 차별화: Topo-I 페이로드, bystander activity

## 사전 공개 임상
- **HERIZON-BTC-01 NCT04466891 Phase 2b**: ORR 41.3%, mDoR 14.9개월, mOS 15.5개월 → FDA 가속승인 근거
- **HERIZON-GEA-01 NCT05152147 Phase 3**: PFS 12.4개월 vs 8.1개월 (통계 유의), OS 26.4 vs 19.2개월 (Arm A) → sBLA Priority Review
- **ZW191 Phase 1 (AACR 2026)**: PROC cORR 61% (6.4~9.6mg/kg), DCR 94%, mPFS 7.6개월

## Modality
**Ziihera (zanidatamab)**
HER2 biparatopic bispecific IgG1-like 항체. HER2 ECD2(dimerization domain) + ECD4(juxtamembrane) 동시 결합. ADCC/CDC 이펙터 기능 보유.

## MOA
- 타겟: HER2 ECD2 + HER2 ECD4 — 동일 분자의 두 에피토프 동시 결합 (biparatopic)
- 기전: ECD2 결합 → HER2-HER3 이형이량체화 억제. ECD4 결합 → 리간드 독립적 신호 차단. Biparatopic 결합 → 강화된 HER2 내재화·수용체 하향조절. IgG1 Fc → ADCC·ADCP. HER2 클러스터링 → C1q 결합 촉진 → CDC.
- 병태생리: HER2 과발현 → PI3K/AKT/mTOR·RAS/MAPK 경로 지속 활성화 → 종양 증식·생존. BTC에서 HER2 증폭은 eCCA·GBC에서 특히 빈번.
