# PDS Biotechnology Corporation (PDSB)

## 회사 개요
Versamune 기반 T세포 활성화 면역치료 플랫폼을 보유한 임상단계 항암 바이오텍. 리드 자산 PDS0101(Versamune HPV)을 HPV16 양성 재발성·전이성 두경부 편평세포암(HNSCC)에서 면역관문억제제(pembrolizumab) 병용으로 개발하며, IL-12 융합 ADC인 PDS01ADC를 다수 적응증의 표준요법 병용 Phase 2로 개발한다. (Versamune는 지질 나노입자 기반 펩타이드 항원 전달 면역치료 플랫폼.)

## 카탈리스트
- Phase 3 VERSATILE-003(NCT06790966) topline 리드아웃: H2 2026 예상. 규제 신청·파트너십 협상의 핵심 카탈리스트.
- VERSATILE-003 프로토콜 개정(2026-02): PFS를 중간분석 1차 평가변수로 추가해 가속승인 경로 추진, mOS는 정식승인용 1차 평가변수 유지. 등록 환자 수 축소로 비용·기간 단축. FDA 30일 검토기간 무이의 경과.

## 플랫폼
- Versamune: 지질 나노입자 기반 펩타이드 항원 전달로 항원특이 CD8+ T세포 활성화를 유도하는 면역치료 플랫폼.
- PDS01ADC: IL-12를 항체에 융합한 ADC. 종양 내 IL-12 전달로 면역세포 활성을 증폭, PDS0101·관문억제제와 삼중병용으로 개발.

## 적응증
- PDS0101 + pembrolizumab · 1차 HPV16 양성 재발성/전이성 HNSCC · Phase 3(VERSATILE-003, NCT06790966) · Versamune HPV16 항원특이 T세포 + PD-1 관문억제.
- PDS01ADC · 전이성 대장암(mCRC) · Phase 2(표준요법 병용) · 등록 완료. Stage 1에서 ORR 77.8%(7/9), 약 24개월 생존율 ~85% 보고.
- PDS01ADC · 전이성 거세저항성 전립선암(mCRPC) · Phase 2 · 모집 진행 중. 초기 데이터 mPFS 9.6개월, PSA 중앙값 40% 감소 보고.

## 임상 디자인
- VERSATILE-003(NCT06790966): PDS0101+pembrolizumab vs pembrolizumab 단독, 무작위 공개라벨 Phase 3, 1차 HPV16 양성 재발성/전이성 HNSCC.
- 선행 Phase 2 VERSATILE-002(NCT04260126): PDS0101+pembrolizumab 병용에서 mOS 39.3개월 보고가 Phase 3 근거.

## 파트너
