# aTyr Pharma, Inc. (ATYR)

## 회사 개요
tRNA 합성효소(HARS) 유래 면역조절 바이오로직 efzofitimod(NRP2 표적 Fc-융합단백)를 간질성 폐질환에서 개발하는 바이오텍. 리드 적응증은 폐 사르코이드증(Phase 3 EFZO-FIT 완료)과 전신경화증 관련 ILD(SSc-ILD, Phase 2 EFZO-CONNECT). 항체가 아니라 융합단백 기반 면역조절제다.

## 카탈리스트
- Phase 2 EFZO-CONNECT(SSc-ILD) 등록 완료 목표: 2026년 상반기(1H2026). 2025년 2분기 고무적 중간분석 발표.
- 폐 사르코이드증: Phase 3 EFZO-FIT 1차 EP 미달 후 FDA Type C 미팅(2026-04경) 진행, 2026년 6월 글로벌 신규 Phase 3 IND 제출 예정.
- FDA: SSc-ILD에 대해 Fast Track 지정 획득.
