# Soligenix, Inc. (SNGX)

## 회사 개요
극초소형 희귀질환 바이오텍. 피부 림프종(CTCL)용 HyBryte의 pivotal Phase 3 FLASH2 시험이 무용성(futility)으로 중간분석 후 중단(2026-04-28)되면서, 선천면역 조절(IDR) 펩타이드 SGX945(dusquetide)로 피벗 중이다. SGX945는 Behcet's Disease 구강궤양 대상 Phase 2a에서 생물학적 효능 신호(apremilast/Otezla Phase 3 디자인에 준함)를 입증했고, UK PIM 지정(2026-03-10)과 EU 희귀의약품 지정(2026-03-26)을 받았다.

리드 자산: SGX945/dusquetide (innate defense regulator peptide/IDR), Behcet's Disease 구강궤양, Phase 2a 완료 → placebo-controlled Phase 2 준비.

## 카탈리스트
가정용(home-use) 제형 개발 완료 후 placebo-controlled Phase 2 개시를 2026년 하반기로 안내. → 2026-H2.

## 적응증
- Behcet's Disease 구강궤양 (SGX945 리드)
- CTCL (HyBryte — FLASH2 무용성 중단)
