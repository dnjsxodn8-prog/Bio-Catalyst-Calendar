# Cognition Therapeutics, Inc. (CGTX)

## 회사 개요
Cognition Therapeutics는 중추신경계 퇴행성 질환을 겨냥하는 임상단계 바이오 기업이다. 선도물질 zervimesine(CT1812)은 경구 1일 1회 복용하는 시그마-2(sigma-2) 수용체 조절제(저분자)로, 알츠하이머병과 루이소체 치매(DLB)에서 손상된 경로를 조절하는 것을 목표로 한다. 알츠하이머병·루이소체 치매에서 Phase 2를 완료했으며, 2025년 12월 CTAD 학회에서 알츠하이머병 Phase 3 계획을 발표하고, 2026년 5월 FDA와 DLB(정신증) 등록 임상 협의를 마쳤다.

## 카탈리스트
- DLB 정신증 등록(registrational) 임상: 2026-05-20 FDA 미팅 완료, "DLB psychosis 개발 경로 확보"라고 발표. 정식 미팅 미닛은 2026년 6월 수령 예상. 세부 디자인·엔드포인트는 미공개.
- 알츠하이머병 Phase 3: End-of-Phase 2 미팅 후 등록 프로그램 설계 발표(CTAD 2025-12), EMA 정렬 미팅 2026년 2월 추진.
- Phase 2 START 시험(초기 알츠하이머/MCI, 545명) topline 데이터: 2027년 예상.

## 플랫폼
- 시그마-2(sigma-2/TMEM97) 수용체 조절 저분자 플랫폼. 시그마-2 수용체 복합체를 통해 아밀로이드-베타 올리고머 등 독성 단백질이 시냅스에 미치는 손상 경로를 차단하는 것을 기전으로 함.

## 적응증
- zervimesine(CT1812) · 알츠하이머병 · Phase 2 완료 → Phase 3 계획 · 시그마-2 수용체 조절(경구 저분자)
- zervimesine(CT1812) · 루이소체 치매(DLB, 정신증 중심) · Phase 2 완료 → 등록 임상 추진 · 동일 기전
- zervimesine(CT1812) · 초기 알츠하이머병/MCI(START 시험) · Phase 2 진행 중 · 동일 기전

## 임상 디자인
- SHINE (알츠하이머병 Phase 2): primary endpoint = 6개월 안전성·내약성(충족), 효능은 탐색적. 경증~중등도 환자 중 baseline plasma p-tau217 낮은 하위군에서 ADAS-Cog11 기준 placebo 대비 인지저하 95% 둔화 신호 보고(CTAD 2025).
- 알츠하이머병 Phase 3 제안 디자인: plasma p-tau217 낮은 바이오마커-enriched 집단에서 6개월 randomized 시험 2건, 100mg 경구 zervimesine vs placebo 1:1, primary efficacy = iADRS, open-label extension 포함(FDA EOP2 기반).
- SHIMMER (DLB Phase 2, NCT05225415): 경증~중등도 DLB 130명, zervimesine(100mg 또는 300mg) 또는 placebo 1일 1회 6개월. primary endpoint = 안전성·내약성(충족), 효능 탐색적. NPI(신경정신 증상)·인지 변동·일상생활동작(ADL) 개선 신호, AD/PD 2026에서 NPI-12 기준 placebo 대비 86% 둔화 보고. Alzheimer's & Dementia 게재.
- START (초기 알츠하이머/MCI Phase 2, NCT05531656): NIA/NIH 연구비 지원, 545명, topline 2027 예상.

## 파트너
없음 (독자 개발). 단, 알츠하이머병·DLB Phase 2 시험은 미국 국립노화연구소(NIA/NIH)의 연구비 지원으로 진행(상업 라이선스 파트너십은 아님). START 시험은 $81M 규모 NIA 그랜트 지원.
