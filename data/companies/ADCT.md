# ADC Therapeutics (ADCT)

## 카탈리스트
LOTIS-5 Phase 3 top-line — ZYNLONTA + rituximab 2차+ DLBCL, PFS 주요 끝점 결과 (2026 Q2)

## 회사 개요
PBD(pyrrolobenzodiazepine) 이량체 기반 ADC 플랫폼 전문기업으로 ZYNLONTA(loncastuximab tesirine)를 미국 FDA 가속승인(3차+ DLBCL, 2021)으로 상업화. LOTIS-5 Phase 3 확증 시험(2L+ DLBCL, rituximab 병용, NCT04384484) top-line이 2026 Q2 예정. 성공 시 sBLA 제출 및 정식 승인·2L 라벨 확장 목표. 현금 $261.3M, 런웨이 2028년+.

## 매출
FY2025 ZYNLONTA 순매출 $73.6M (FY2024 $69.3M 대비 +6%); 현금 $261.3M

## 플랫폼
- PBD 이량체 ADC 기술 (DNA 가교 결합 세포독성)
- ZYNLONTA (loncastuximab tesirine-lpyl, anti-CD19 ADC)

## 적응증
재발/불응성 DLBCL (승인), 여포림프종, 기타 B세포 림프종(임상 중)

ZYNLONTA (loncastuximab tesirine, 3차+ DLBCL, FDA 가속승인 2021)

## 임상 디자인
- 디자인: Randomized, open-label, Phase 3 (LOTIS-5, NCT04384484)
- Arms: Lonca-R (loncastuximab tesirine + rituximab) vs R-GemOx (rituximab + gemcitabine + oxaliplatin)
- N: ~330명 (1:1 무작위 배정); 자격: ≥1차 이전 치료, DLBCL/HGBCL 진단
- Phase: Phase 3 (등록 완료, 데이터 성숙 2026 Q2)
- Primary endpoint: PFS (무진행 생존기간) — ~262 PFS 이벤트 시 분석
- Secondary endpoint: OS, ORR, CR율, DoR, 안전성

## 타겟 질환
**재발/불응성 미만성 대B세포 림프종 (Relapsed/Refractory DLBCL)**

비호지킨 림프종 중 가장 흔한 공격성 아형. 1차 R-CHOP 후 재발·불응 시 예후 불량.

- 환자 수: 미국 신규 DLBCL ~24,000명/년; 2차+ 치료 환자 약 40%
- 병리: B세포 림프종 CD19 보편 발현. R-CHOP 실패 후 2L 치료 옵션 제한 (CAR-T 비적격 환자 다수).

## 기존 치료제
R-CHOP (1차), CAR-T(axi-cel, tisa-cel, liso-cel — 2차 옵션, 적격 환자 한정), Polivy(포라투주맙 vedotin+R-CHP). ZYNLONTA 병용이 CAR-T 비적격 2L+ 환자의 대안.

## 사전 공개 임상
- **LOTIS-2 Phase 2 (단독요법) · NCT03589469 · Phase 2**
  결과: ORR 48.3%, CR 24.1% (3차+ DLBCL) → FDA 가속승인 근거
- **LOTIS-7 Phase 1b (+ glofitamab) · 진행 중**
  결과 (ASH 2025): 49명 평가 가능 환자 CR율 77.6%; ~100명 등록 완료 목표 (2026 상반기), 전체 데이터 2026년 말 학회 발표 예정

## Modality
**ZYNLONTA (Loncastuximab tesirine-lpyl)**
항-CD19 PBD 이량체 ADC (type: ADC)

인간화 항-CD19 IgG1 mAb에 PBD 이량체 독소(SG3199) linker 결합. 카텝신 절단형 링커.

## MOA
- 타겟: CD19 (B세포 표면 항원, DLBCL 보편 발현)
- 기전: 항-CD19 mAb가 B세포 CD19에 결합 → 수용체 매개 내재화 → 리소솜 내 카텝신이 링커 절단 → PBD 이량체(SG3199) 방출 → DNA 소홈에 공유 가교 결합 → DNA 복제·전사 불가 → 세포 사멸. PBD 독소는 bystander effect 최소(막 투과성 낮음).
- 병태생리: CD19는 거의 모든 B세포 림프종에서 발현. PBD 가교는 DNA 절단 없이 비가역적 손상으로 MDR 내성 극복 가능.
