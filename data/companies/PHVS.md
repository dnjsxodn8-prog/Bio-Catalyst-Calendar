# Pharvaris (PHVS)

## 카탈리스트
Deucrictibant (PHVS416) — 유전성 혈관부종(HAE) Phase 3 CHAPTER-3 중간 데이터 / 최종 결과 — 2026년 하반기 예상

## 회사 개요
Pharvaris는 유전성 혈관부종(HAE) 대상 경구 브래디키닌 B2 수용체 길항제(deucrictibant) 개발에 집중하는 임상 단계 바이오텍. 네덜란드 기반.

기존 HAE 치료제(피하 주사, 정맥 주사)의 불편함을 극복한 경구 제형이 핵심 차별점. Phase 3 CHAPTER 프로그램 진행 중.

## 임상 디자인
- 디자인: Randomized, double-blind, placebo-controlled
- Arms: Deucrictibant 40mg QD vs. 위약 (예방 투여)
- N: 약 140명 (HAE-1/2 성인, 반복 발작)
- Phase: Phase 3 (CHAPTER-3, 예방 투여)
- Primary endpoint: HAE 발작 빈도 감소율 (위약 대비)
- Secondary endpoint: 발작 중증도, 삶의 질, 안전성

## 타겟 질환
**유전성 혈관부종 1·2형 (Hereditary Angioedema Type 1/2, HAE)**

C1-esterase inhibitor 결핍 또는 기능 이상으로 브래디키닌 과다 생성 → 피부·점막·내장 반복 부종. 후두 발작 시 생명 위협.

- 환자 수: 미국 약 8,000명, 글로벌 약 50,000명 (1:50,000 유병률)
- 병리: C1-INH 결핍 → 칼리크레인-키닌계 항진 → 브래디키닌 과잉 → B2R 활성화 → 혈관 투과성 증가 → 부종

## 기존 치료제
- Takhzyro (lanadelumab, Takeda) — 피하 주사 예방
- Haegarda (C1-INH, CSL) — 피하 주사 예방
- Garadacimab (Kalvista/AZ) — 경구 예방 (승인 심사 중)
- Berotralstat (KalVista, BioCryst) — 경구 예방 (승인)
Deucrictibant은 브래디키닌 B2R 직접 길항으로 칼리크레인 억제와 다른 기전.

## 사전 공개 임상
- **CHAPTER-1 Phase 2 (급성 치료) · NCT04618393 · Phase 2**
  결과: Positive (2022) — 발작 시간 유의 단축, 위약 대비 통계적 유의
- **CHAPTER-2 Phase 2 (예방) · NCT04870801 · Phase 2**
  결과: Positive (2023) — 발작 빈도 67% 감소 (위약 대비)
- **CHAPTER-3 Phase 3 (예방) · NCT05874895 · Phase 3**
  결과: 진행 중 — 2026년 하반기 데이터 예상

## Modality
**Deucrictibant (PHVS416)**
Bradykinin B2 receptor antagonist (type: Small Molecule)

경구 복용 1일 1회. 중수소화(deuterium) 치환으로 대사 안정성 향상 (d4-icatibant 유도체).

## MOA
- 타겟: 브래디키닌 B2 수용체 (BDKRB2)
- 기전: 브래디키닌의 B2R 결합 경쟁적 차단 → 혈관 내피 투과성 증가 신호 억제 → 부종 형성 차단. Icatibant(피하주사) 경구 유도체로 약동학 개선.
- 병태생리: HAE 발작 시 브래디키닌 급격한 증가가 B2R 통한 혈관 투과성 항진의 최종 경로. B2R 직접 차단이 칼리크레인 또는 C1-INH 보충과 달리 최하류(downstream) 표적.
