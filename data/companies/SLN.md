# Silence Therapeutics (SLN)

## 카탈리스트
Divesiran Phase 2 SANRECO (다혈구증) 데이터 — 2026년 3Q 예정

## 회사 개요
Silence Therapeutics는 영국 기반 GalNAc-siRNA 플랫폼(mRx1) 전문 RNAi 기업. 선도 물질 zerlasiran(SLN360)은 Lp(a) 저하 siRNA(심혈관 위험). 차세대 divesiran(SLN124)은 TMPRSS6 타겟 siRNA로 다혈구증(PV) Phase 2 SANRECO 등록 완료, 2026년 3Q 데이터 예정. AstraZeneca 파트너십(SLN312) 종료 후 권리 반환 예정.

## 매출
수익 없음. R&D 단계.

## 플랫폼
- mRx1 GalNAc-siRNA 플랫폼 (간 타겟팅 siRNA 전달)
- 피하 주사, 장기 지속 효과

## 적응증
Cardiovascular (Lp(a) 상승, 심혈관 위험), Rare Disease (다혈구증 PV)

## 파트너
- AstraZeneca (SLN312, dyslipidemia — Phase 1 이후 개발 종료, 권리 반환 예정)

## 매출 구조
수익 없음. 파트너십 마일스톤 기반.

## 자체 판매
아니오 - 임상 단계

## 상업화 제품
없음

## 임상 디자인
- SANRECO Phase 2 (Divesiran/SLN124): 무작위, 이중맹검, 위약 대조
  - 대상: 사혈 의존성 다혈구증(PV) 환자 48명
  - 1차 평가변수: 18~36주 사이 사혈 없이 헤마토크릿 <45% 유지 비율
  - 데이터 예정: 2026년 3Q
- Zerlasiran Phase 2 완료: Lp(a) 유의미 저하, Phase 3 파트너 물색 중

## 타겟 질환
**다혈구증 (Polycythemia Vera, PV)**

JAK2 변이로 적혈구 과다 생성 → 혈전·출혈 위험 증가.

- 환자 수: 미국 약 65,000~95,000명
- 병리: 헤파시딘 경로 과활성 → 철 흡수 과잉 → 적혈구 과다 생성. TMPRSS6 억제로 헤파시딘 회복 → 적혈구 억제.

## 기존 치료제
사혈, 하이드록시우레아, ruxolitinib(Jakafi). Divesiran은 TMPRSS6 siRNA로 철 항상성 조절 → 사혈 의존성 탈피 목표.

## 사전 공개 임상
- **Divesiran SANRECO Phase 2 · 등록 완료**
  데이터 예정: 2026년 3Q.
- **Zerlasiran Phase 2 (Lp(a)) · 완료**
  결과: Lp(a) 유의미 저하. Phase 3 진행 위해 파트너 물색 중. 업계 첫 Lp(a) CVOT 2026년 결과 예정.

## Modality
**Divesiran (SLN124)**
GalNAc-conjugated siRNA targeting TMPRSS6 mRNA (type: RNA/siRNA)

GalNAc 결합 siRNA로 간 세포 선택적 전달. TMPRSS6 mRNA 절단 → 헤파시딘 발현 증가 → 적혈구 생성 억제.

## MOA
- 타겟: TMPRSS6 (matriptase-2) mRNA
- 기전: siRNA가 RISC 복합체와 결합 → TMPRSS6 mRNA 서열 특이적 절단 → TMPRSS6 단백질 감소 → 헤파시딘 억제 해제 → 헤파시딘 상승 → 철 제한 → 적혈구 과다 생성 억제
- 병태생리: PV에서 JAK2 활성으로 과다 적혈구 생성. TMPRSS6 억제가 하류에서 적혈구 생성 브레이크.
