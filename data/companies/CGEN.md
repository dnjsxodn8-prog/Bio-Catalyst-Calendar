# Compugen (CGEN)

## 카탈리스트
COM701(PVRIG 차단항체) MAIA-ovarian Phase 2 초기 데이터 — 2026년 내; AZ rilvegostomig Phase 3 데이터 다수 발표

## 회사 개요
이스라엘 소재 계산생물학 기반 면역항암 타깃 발견 전문 바이오텍. 독자 알고리즘으로 PVRIG, TIGIT 등 면역 억제 수용체를 발굴. AstraZeneca가 COM902(anti-TIGIT) 기반 rilvegostomig를 10개 Phase 3 시험에서 독자 개발 중. 2025년 AZ로부터 rilvegostomig 미래 로열티 선급금 $65M 수령(최대 $90M) → 2029년까지 런웨이.

자체 리드 COM701(anti-PVRIG)은 MAIA 난소암 시험에서 bevacizumab+niraparib 병합 요법 탐색 중.

## 매출
FY2025 ~$72.8M (+161% YoY, AZ 로열티 선급금 포함); FY2024 ~$27.9M

## 플랫폼
- COM701 (anti-PVRIG mAb): Phase 1/2 난소암(MAIA 시험); CM-24(anti-CEACAM1) 병합 탐색
- COM902 기반 rilvegostomig (anti-TIGIT + anti-PD-1, AZ 개발): Phase 3 ×10
- 계산생물학 타깃 발견 플랫폼 (면역 억제 경로 신타깃 발굴)

## 적응증
난소암, 자궁내막암, 기타 고형암 (면역항암 병합요법)

## 파트너

## 임상 디자인
- 약물: COM701 (anti-PVRIG 단일클론항체)

- Secondary endpoint: DCR, PFS, DOR

## 타겟 질환
**난소암 (재발성/불응성 — Platinum-resistant 포함)**

상피성 난소암은 PD-1/PD-L1 단독 반응률 낮음; PVRIG-PVRL2 경로가 추가적 면역 억제 축으로 작용.

- 환자 수: 미국 신규 ~20,000명/년; 백금 불응성 환자 미충족 수요 높음
- 병리: PVRL2(PVR-related protein 2) 과발현 종양이 T세포 표면 PVRIG에 결합 → T세포 기능 억제; TIGIT와 다른 독립적 억제 경로

## 기존 치료제
bevacizumab(Avastin, Roche), PARP 억제제(olaparib·niraparib·rucaparib), pembrolizumab(일부 적응증). COM701은 PVRIG라는 신규 면역 억제 경로를 표적하여 PD-1·CTLA-4 불응 환자에서 추가 반응 기대.

## 사전 공개 임상
- **COM701 단독 및 nivolumab 병합 · Phase 1 · NCT04570839**
  결과: 2023 AACR/ESMO — Phase 1 용량 증량: 안전성 양호, PVRIG+ 종양에서 T세포 기능 회복 확인; ORR(전체) 약 10-15% (소규모 코호트); PVRL2 고발현 환자에서 반응률 높은 경향
- **Rilvegostomig (COM902 기반, AZ) · TIGIT+PD-1 이중 차단 · Phase 3**
  NCT: 다수 (AZ MATTERHORN 등); 2025년 첫 Phase 3 중간 데이터 예정

## Modality
**COM701**
단일클론항체 (Monoclonal Antibody) — PVRIG(CD112R) 차단항체

IgG4 기반 mAb. T세포·NK세포 표면 PVRIG에 결합하여 PVRL2(리간드) 결합 차단. 면역 억제 시그널 해제.

## MOA
- 타겟: PVRIG (PVR-related Ig domain containing protein; CD112R) — T세포 및 NK세포 표면 억제 수용체
- 기전: COM701이 PVRIG에 결합 → PVRL2 결합 차단 → PVRIG/PVRL2 억제 시그널 해제 → T세포·NK세포 활성화 회복
- TIGIT와의 관계: TIGIT와 PVRIG는 독립적이지만 상호보완적 억제 경로 → PD-1+TIGIT+PVRIG 삼중 차단으로 시너지 효과 가설
- 병태생리: 난소암 등 고형암에서 PVRL2 과발현 → PVRIG 매개 T세포 기능 억제 → 면역 회피
