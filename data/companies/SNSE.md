# Sensei Biotherapeutics, Inc. (SNSE)

> 2026-06-16부로 사명을 Faeth Therapeutics, Inc.로 변경하고 나스닥 티커를 SNSE → FTH로 전환했다.

## 회사 개요
임상단계 종양학 기업. 기존 VISTA 표적 항체 solnerstotug를 중단한 뒤, 2026년 2월 17일 종양 대사 표적 치료제 개발사 Faeth Therapeutics를 인수($200M 사모 동반)하며 사업을 전환했다. 인수로 확보한 리드 자산은 PIKTOR — serabelisib(선택적 PI3K-alpha 억제제)과 sapanisertib(mTORC1/mTORC2 억제제)의 경구 병용으로 PI3K/AKT/mTOR 경로를 다중 노드에서 동시 차단하는 소분자 조합이다. 2026년 6월 16일 사명을 Faeth Therapeutics로 변경하고 티커를 FTH로 전환했다. 동시에 Faeth 공동창업자 Anand Parikh가 회장·사장·CEO, Brian Stephenson(Ph.D., CFA)이 CFO로 선임됐고, 이사회에 전 FDA 국장 Stephen M. Hahn(M.D.)과 Saira Ramasastry가 합류했다.

## 카탈리스트
- 2026-H2: 2차 치료 진행성 자궁내막암 대상 PIKTOR(+paclitaxel) Phase 2 topline 데이터 예상.
- 2027: HR+/HER2- 진행성 유방암 Phase 1b/2 중간 데이터 예상(첫 환자 투약 2026-04).

## 플랫폼
- MetabOS: 유전체·전사체·종양미세환경 데이터를 통합해 종양의 대사 의존성을 식별·표적하는 데이터 기반 플랫폼. 자궁내막암 Phase 2에는 인슐린 억제 식이(insulin-suppressing diet) 서브스터디를 병행해, PI3K 억제 시 발생하는 혈당·인슐린 상승을 억제함으로써 안전성·효능 개선 가능성을 평가한다.

## 적응증
- serabelisib + sapanisertib (PIKTOR) + paclitaxel · 진행성/재발 자궁내막암 (2L+, Phase 2, GOG-3111 / FTH-PIK-201) · PI3K-alpha + mTORC1/2 이중 억제. NCT06463028
- serabelisib + sapanisertib (PIKTOR) · HR+/HER2- 진행성/전이성 유방암 (Phase 1b/2, FTH-PIK-101) · PI3K-alpha + mTORC1/2 이중 억제. NCT07558733

## 임상 디자인
- 자궁내막암 Phase 2(NCT06463028, GOG-3111): 다기관·공개·단일군. 백금 기반 항암제와 면역관문억제제 등 선행 전신요법 실패 환자 대상. GOG Foundation 주도.
- 사전 공개 임상(Phase 1b, NCT03154294): paclitaxel + serabelisib + sapanisertib 3제 병용에서 치료 불응성 진행성 자궁내막암 환자 5명 중 ORR 80%(CR 3, PR 1) 보고 — 이 결과가 Phase 2 개시 근거.

## 파트너
- Faeth Therapeutics (인수, 리드 자산 PIKTOR 확보, $200M 사모 동반, 2026-02). 자궁내막암 Phase 2는 GOG Foundation과 공동 수행.

## Modality / MOA
- 전 경구 소분자 병용(serabelisib + sapanisertib). PI3K-alpha와 mTORC1/2를 동시 억제해 PI3K/AKT/mTOR 경로를 복수 노드에서 차단.

## 메모
- 2026-02 Faeth 인수와 동시 $200M 총 조달의 사모 발행. 사업 전환 + 경영진·이사회 전면 개편(전 FDA 국장 Hahn 영입).
- 향후 12개월 핵심 가치 모멘텀은 2026-H2 자궁내막암 Phase 2 topline. Phase 1b 80% ORR(소수 환자)의 Phase 2 재현 여부가 관건.
- 마이크로캡(스크리너 유래). 자금난·저성숙 단계 리스크.
