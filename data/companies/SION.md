# Sionna Therapeutics (SION)

## 카탈리스트
PreciSION CF Phase 2a (SION-719 + Trikafta) 데이터 — 2026년 여름 예정

## 회사 개요
Sionna Therapeutics는 낭포성 섬유증(CF) 치료의 완전한 CFTR 기능 정상화를 목표로 하는 소분자 기업. 선도 물질 SION-719는 최초 NBD1 안정화제(first-in-class). Trikafta(Vertex 표준치료)에 추가하는 병용 전략. PreciSION CF Phase 2a 등록 완료(2026.4.27), 여름 데이터 예정. Vertex 대항마로 주목.

## 매출
수익 없음. 임상 단계.

- NBD1 (Nucleotide Binding Domain 1) 안정화제
- CFTR 모듈레이터 복합 전략

## 적응증
Rare Disease (낭포성 섬유증, Cystic Fibrosis)

## 파트너
없음 (독자 개발)

## 임상 디자인
- PreciSION CF Phase 2a: 무작위, 이중맹검, 위약 대조, 교차 설계
- 대상: F508del 동형접합 성인 CF 환자 (Trikafta 안정 복용 중)
- 1차 평가변수: 땀 염화물(sweat chloride) 변화 (CFTR 기능 바이오마커)
- 안전성·내약성·PK도 평가
- 데이터 예정: 2026년 여름

## 타겟 질환
**낭포성 섬유증 (Cystic Fibrosis, CF)**

CFTR 유전자 변이로 염소이온 채널 기능 소실 → 폐·소화기관 점액 축적 → 반복 감염·폐 손상.

- 환자 수: 미국 약 30,000명, F508del 동형접합 다수
- 병리: CFTR 단백질의 폴딩·안정성 결함 → 기능 상실. NBD1 도메인 불안정이 핵심 병태.

## 기존 치료제
Trikafta (elexacaftor/tezacaftor/ivacaftor, Vertex). SION-719는 Trikafta 위에 추가해 CFTR 기능 추가 개선 목표.

## 사전 공개 임상
- **PreciSION CF Phase 2a (SION-719) · 등록 완료 2026.4.27**
  데이터 예정: 2026년 여름. 전임상에서 완전한 CFTR 기능 정상화 데이터 발표.

## Modality
**SION-719**
First-in-class oral NBD1 stabilizer for CFTR (type: Small Molecule)

CFTR 단백질의 NBD1 도메인 열안정성 강화 → Trikafta와 시너지로 CFTR 기능 추가 복원.

## MOA
- 타겟: CFTR 단백질 NBD1 도메인
- 기전: NBD1 결합 → 도메인 열안정성 향상 → CFTR 폴딩 개선 → 세포 표면 발현 증가 + 채널 개방 확률 상승. Trikafta의 교정자(corrector)/증폭자(potentiator)와 시너지.
- 병태생리: F508del CF에서 CFTR의 NBD1 불안정이 단백질 분해·기능 손실 유발. NBD1 안정화가 Trikafta 한계를 극복할 수 있는 새 레이어.
