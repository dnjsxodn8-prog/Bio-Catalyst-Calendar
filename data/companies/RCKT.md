# Rocket Pharmaceuticals (RCKT)

## 카탈리스트
RP-A501 (Danon병 AAV9 유전자 치료) Phase 1 장기 데이터 및 FDA 허가 신청 전략 발표 (2026 H2 예정); RP-L102 (Fanconi 빈혈 렌티바이러스) Phase 2 데이터

## 회사 개요
Rocket Pharmaceuticals는 렌티바이러스(ex vivo HSC) 및 AAV(in vivo) 유전자 치료 플랫폼을 활용해 희귀 혈액·심장 질환을 치료하는 중소형 바이오텍. 2026년 3월 FDA가 KRESLADI (RP-L401, marnetegragene autotemcel)를 LAD-I (Leukocyte Adhesion Deficiency Type I) 치료제로 승인 — 회사 첫 상업화 제품. Danon병(RP-A501), Fanconi 빈혈(RP-L102), 선천성 적혈구 조혈 이상성 빈혈(PKD, RP-L301) 파이프라인 보유.

KRESLADI 승인으로 상업화 단계 진입. RP-A501 Danon병 데이터가 차기 최대 카탈리스트.

## 임상 디자인
- 디자인: Open-label, single-arm, dose-escalation
- Arms: RP-A501 단일 용량 코호트 (저용량·고용량)
- N: Phase 1 ~10명 (Danon병 소아·청소년 남성)
- Phase: Phase 1 (미국, 소아 Danon병)
- Primary endpoint: 안전성·내약성 (12개월)
- Secondary endpoint: 심근 LAMP2 발현 회복, 심장 기능 지표 (LV EF, 심근 두께), 활동 내성

## 타겟 질환
**Danon병 (Danon Disease)**

LAMP2 (Lysosomal-associated membrane protein 2) 유전자 돌연변이로 인한 X-연관 리소좀 저장 질환. 심근증·근병증·지적 장애 3대 증상. 청소년기 급격한 심장 기능 저하로 심장이식 외 치료법 없음.

- 환자 수: 전 세계 ~1/200,000 희귀도. 남성 환자는 10-20대에 심부전·돌연사 위험 높음.
- 병리: LAMP2 결손 → 자가포식(autophagy) 장애 → 심근세포 내 자가포식소체 집적 → 심근 비대·섬유화 → 심부전

## 기존 치료제
없음 (근본 치료). 심부전 약물, 부정맥 치료기(ICD), 심장이식만 가능. RP-A501은 최초 유전자 치료 시도.

## 사전 공개 임상
- **RP-A501 Phase 1 초기 데이터 · NCT03882437 · Phase 1**
  결과: 2023-2024 ESC/NEJM — 소아 Danon 환자 6명 치료 후 LAMP2 단백질 발현 회복, 심근 두께 감소, LV 기능 개선. 고용량에서 면역반응(면역억제로 관리). 장기 추적 진행.

## Modality
**RP-A501**
AAV9 벡터를 이용한 심장 지향성 in vivo 유전자 치료제. LAMP2B 기능 유전자를 정맥 단회 주사로 심근세포에 전달하여 리소좀 기능 회복.
(Modality 분류: Gene Therapy)

## MOA
- 타겟: LAMP2B (리소좀 막 단백질 2B 아이소폼)
- 기전: AAV9가 심근세포에 선택적 형질도입 → LAMP2B 유전자 발현 → 리소좀 막 무결성 회복 → 자가포식 정상화 → 심근세포 내 자가포식소체 청소 → 심근 기능 회복
- 병태생리: LAMP2 결손 → 자가포식소체(autophagosome) 리소좀 융합 실패 → 세포 폐기물 집적 → 심근세포 비대·사멸 → 제한성 심근증. RP-A501은 근본 원인인 LAMP2B 부재를 교정.
