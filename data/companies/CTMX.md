# CytomX Therapeutics (CTMX)

## 카탈리스트
CX-2051 (Probody ADC, EpCAM 타겟) Phase 1 용량 확장 데이터 (2026 H1 예상) — CX-904는 개발 중단됨(TERMINATED)

## 회사 개요
CytomX Therapeutics는 Probody 기술 기반 조건부 활성화 항체·ADC 개발사. Probody: 마스킹 펩타이드로 항체를 비활성 상태로 유지 → 종양 미소환경의 프로테아제(matriptase 등)에 의해 특이적 활성화. 정상 조직 부작용 최소화, 종양 특이성 극대화. BMS, AbbVie와 대형 파트너십 보유 이력. 현재 독자 파이프라인 EpCAM ADC(CX-2051) 주력.

## 매출
파트너십 마일스톤 수입 약간. 상업화 제품 없음.

## 타겟 질환
**EpCAM 발현 고형암 (대장암, 폐암, TNBC 등)**

EpCAM(Epithelial Cell Adhesion Molecule)은 정상 상피세포 포함 광범위 발현으로 기존 항체 독성 문제. Probody ADC는 종양 특이적 활성화로 이를 해결.

- 환자 수: EpCAM 고발현 대장암(95%), 폐선암(70%), TNBC(50%) 등
- 병리: EpCAM 정상 조직 발현으로 기존 anti-EpCAM 항체(catumaxomab 등) 독성 심각. Probody ADC는 종양 미소환경에서만 활성화.

## 기존 치료제
Catumaxomab (EpCAM/CD3 BiTE, 복강내암 적용). 전신 투여 EpCAM 타겟 제제는 독성 문제로 개발 어려움. CX-2051은 Probody 기술로 치료 지수 개선 목표.

## Modality
**CX-2051**
EpCAM-타겟 Probody ADC (type: ADC)

Probody 마스킹 펩타이드 + anti-EpCAM 항체 + 세포독성 페이로드 접합체. 종양 미소환경의 matriptase(ST14) 프로테아제가 마스킹 펩타이드 절단 → 항체 활성화 → EpCAM 결합 → 내재화 → 페이로드 방출.

## MOA
- 타겟: EpCAM (EPCAM/TACSTD1)
- 기전: Probody 불활성 상태로 전신 순환 → 종양 미소환경 matriptase 절단 → 활성 anti-EpCAM ADC → EpCAM 결합 → 내재화 → 세포독성 페이로드 방출 → 암세포 사멸
- 병태생리: EpCAM 광범위 발현으로 전신 독성 우려를 종양 국소 활성화로 해결. 기존 EpCAM 치료제의 치료지수 한계를 Probody ADC로 극복.
