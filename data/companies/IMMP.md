# Immutep (IMMP)

## 카탈리스트
Eftilagimod alpha (efti, IMP321) TACTI-004 Phase 3 중간 결과 (NSCLC 1차 치료) — 2026년 하반기 예정

## 회사 개요
Immutep는 호주 시드니 기반 면역항암 임상 단계 바이오텍. LAG-3 계열 면역조절 단백질 기반 파이프라인 보유. 리드 자산 eftilagimod alpha(efti)는 재조합 LAG-3 단백질(MHC II 작용제)로 항원제시세포(APC)를 활성화해 항종양 면역 증강. 기존 PD-1/LAG-3 항체와 달리 LAG-3의 작용제(agonist) 기전으로 차별화. AstraZeneca, Merck 등 빅파마와 임상 협력.

## 매출
상업 매출 없음. 협력 계약 수익 + 그랜트. FY2024 현금 보유 ~A$70M.

## 임상 디자인
- 시험명: TACTI-004 (NCT04428047)
- Phase: Phase 3
- N: 약 480명 (PD-L1 TPS≥1% NSCLC, 1차 치료)
- 디자인: Randomized, double-blind, placebo-controlled. Efti + Pembrolizumab vs Placebo + Pembrolizumab
- Primary endpoint: 무진행생존기간(PFS), 전체생존기간(OS) [공동 1차)
- Secondary endpoint: ORR, DOR, 안전성

## 타겟 질환
**비소세포폐암 (NSCLC) 1차 치료**

PD-L1 발현 NSCLC에서 Keytruda 단독 표준요법의 한계(ORR ~45%)를 efti 병용으로 극복 시도.

- 환자 수: 미국 NSCLC 신규 환자 약 23만 명/년, PD-L1 TPS≥1% 약 70%
- 병리: 종양 미세환경의 APC 기능 저하, T세포 프라이밍 부족 → efti로 APC 활성화 → 항종양 T세포 반응 증폭

## 기존 치료제
Keytruda (pembrolizumab) 단독 또는 화학요법 병용이 표준. PD-L1 고발현(TPS≥50%)에서 Keytruda 단독 반응률 ~45%. Efti 병용으로 추가 반응 유도 목표.

## 사전 공개 임상
- **TACTI-002 Phase 2 (NCT03625323) · Efti + Pembrolizumab · NSCLC/HNSCC**
  결과: NSCLC 2차 치료 코호트 ORR 36%, DCR 62%. 2023 ESMO 발표. HNSCC 1차 코호트 ORR 43%. 내약성 양호.
- **AIPAC Phase 2b (NCT02614833) · Efti + Paclitaxel · 전이성 유방암**
  결과: PFS 개선 신호(HR 0.73), 통계적 유의성 미달. 탐색적 PD-L1 양성군에서 신호.

## Modality
**Eftilagimod alpha (efti, IMP321)**
Recombinant LAG-3 Ig Fusion Protein (Biologic)

LAG-3 세포외 도메인을 IgG1 Fc에 융합한 재조합 단백질. 피하 주사. MHC Class II에 결합하여 수지상세포·대식세포 활성화 → T세포 프라이밍 증폭.

## MOA
- 타겟: MHC Class II (HLA-DR 등) on Antigen-Presenting Cells (APC)
- 기전: Efti가 APC의 MHC Class II에 고친화도 결합 → APC 활성화(성숙) → 사이토카인 분비(IL-12 등) 증가 → CD4/CD8 T세포 프라이밍·증폭 → 항종양 면역 증강. PD-1 억제제와 보완적 기전으로 상가(additive) 효과 기대.
- 병태생리: 종양 미세환경에서 APC 기능 저하로 T세포 활성화 불충분 → Efti로 APC 활성화 → PD-1 억제제 효과 증폭.

## 메모
LAG-3 작용제(agonist)라는 독특한 기전. PD-1 억제제와 상보적 메커니즘. TACTI-004 Phase 3이 핵심 카탈리스트. 소형 바이오텍으로 Phase 3 실패 시 유동성 위험. 빅파마 파트너링 딜이 주가 상방 트리거.
