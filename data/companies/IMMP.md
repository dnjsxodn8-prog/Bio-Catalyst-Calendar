# Immutep Limited (IMMP)

## 회사 개요
LAG-3 표적 면역요법을 개발하는 호주 임상단계 바이오텍. 항암 리드였던 eftilagimod alfa(efti, IMP321, 가용성 LAG-3 융합단백질·MHC class II agonist / APC activator)의 1L NSCLC Phase 3 TACTI-004(KEYNOTE-F91)는 2026년 3월 무용성 분석 후 중단됨(약 170명 시점, efti arm이 대조군 대비 열위·INSIGHT-003 결과 대비 "예상외 부진"). 이후 파이프라인 무게중심이 자가면역용 first-in-class LAG-3 agonist 항체 IMP761로 이동.

## 카탈리스트
- IMP761(LAG-3 agonist 항체, 자가면역): 2026-06-04 EULAR(런던)에서 first-in-human Phase 1 중간 데이터 발표 — 단회상승용량(SAD) 1차 EP(안전성·내약성) 충족, 모든 용량 안전·양호 내약. 7mg/kg에서 통계적으로 유의한 약력학 활성(국소 염증반응 감소·T세포 활성 억제, 피부혈류 개선 p=0.029). PK상 4주 1회 투여 가능.
- IMP761 다중상승용량(MAD): 2개 용량 평가 중, **완료 시점 3Q CY2026 예상**. 추가 임상 업데이트 2H CY2026 예정, Phase 2(류마티스 관절염 등) 진입 가능성.
- efti(STS): EFTISARC-NEO가 1차 목표 충족, 2026-04 연조직육종(STS) 대상 FDA Orphan Drug Designation 획득.
- ASCO 2026: 2026-05-30 efti 포스터(Abstract #2569, 발표자 Martin Forster) — 전이암 다수 후기연구 통합, MHC class II 경유 림프구 활성화와 OS 개선 연관(TACTI-004 데이터는 제외).
- efti TACTI-004 root-cause 분석: 2026년 3월 IDMC 무용성 판정 후 진행 중, Q3 CY2026까지 연장 가능. efti 전체 개발 프로그램 영향 검토 중.

## 플랫폼
LAG-3(Lymphocyte Activation Gene-3) 면역 체크포인트를 양방향으로 공략:
- Agonist 경로(IMP761) — LAG-3 작용으로 T세포 활성 억제 → 자가면역질환
- APC 활성화 경로(efti, IMP321) — 가용성 LAG-3·MHC class II 결합으로 항원제시세포 활성화 → 항암

## 적응증
- efti(eftilagimod alfa, IMP321) · 1L NSCLC 등 고형암 · Phase 3 TACTI-004 중단(2026-03) · 가용성 LAG-3 융합단백질, MHC class II agonist
- efti · 연조직육종(STS, EFTISARC-NEO) · 1차 목표 충족, FDA ODD(2026-04)
- IMP761 · 자가면역질환(류마티스 관절염 등) · Phase 1 진행(SAD 완료·MAD 진행) · first-in-class LAG-3 agonist 항체
