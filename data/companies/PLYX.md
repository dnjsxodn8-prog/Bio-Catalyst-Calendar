# Polaryx Therapeutics (PLYX)

## 카탈리스트
PLX-300 (트레할로스 IV, CLN3 Batten병) Phase 2/3 데이터 — 2026년 하반기 예상

## 회사 개요
Polaryx Therapeutics는 Batten병(신경 세로이드 리포푸신증, NCL) 등 희귀 신경퇴행성 질환 대상 트레할로스(trehalose) 기반 치료제 개발 바이오텍.

트레할로스가 자가포식(autophagy) 유도를 통해 세포 내 이상 단백질·지질 축적 제거 기전 활용. PLX-300은 정맥 내 트레할로스 제제.

## 임상 디자인
- 디자인: Open-label, dose-escalation (Phase 2/3 전환형)
- Arms: PLX-300 정맥 주사 단일군 (용량 탐색 후 확장)
- N: 약 20~30명 (CLN3 Batten병 소아·청소년)
- Phase: Phase 2/3
- Primary endpoint: 신경 기능 유지 지수 (Hamburg Motor and Language Scale, HMLS)
- Secondary endpoint: MRI 뇌 위축 진행 속도, 삶의 질, 안전성

## 타겟 질환
**CLN3 Batten병 (Neuronal Ceroid Lipofuscinosis type 3)**

CLN3 유전자 돌연변이로 세포 내 리포푸신 축적 → 신경세포 사망 → 시력 상실, 발작, 인지·운동 저하, 조기 사망.

- 환자 수: 글로벌 Batten병 전체 약 14,000명; CLN3 가장 흔한 청소년형 (~6,000명 추정)
- 병리: CLN3 결핍 → 리소좀 기능 장애 → 세로이드 리포푸신 축적 → 망막·뇌 신경세포 진행성 사망

## 기존 치료제
- BioMarin의 cerliponase alfa (Brineura) — CLN2 Batten병 승인 (CLN3와 다른 유형)
- CLN3형 승인 치료제 없음 (언메트 니즈 매우 높음)

## 사전 공개 임상

  결과: 안전성 확인 (2022); IV 트레할로스 자가포식 유도 biomarker 개선 신호
- **PLX-300 Phase 2/3 (CLN3) · 진행 중**
  결과: 진행 중 — 2026년 하반기 중간 데이터 예상

## Modality
**PLX-300 (트레할로스 IV)**
Trehalose intravenous formulation (type: Small Molecule)

비환원 이당류 트레할로스 정맥 주사 제제. 경구 투여 시 장내 흡수 불량 → 정맥 투여로 CNS 도달.

## MOA
- 타겟: 자가포식(Autophagy) 경로 (mTOR 비의존)
- 기전: 트레할로스가 TFEB(Transcription Factor EB) 핵 이동 촉진 → 리소좀 생합성·자가포식 관련 유전자 발현 증가 → 세포 내 이상 단백질·리포푸신 제거 촉진. mTOR 경로 비의존적이어서 영양 상태 무관.
- 병태생리: CLN3 결핍 리소좀 기능 저하로 세로이드 축적. 자가포식 강화가 잔여 리소좀 기능을 통해 독성 물질 일부 제거. 신경세포 사망 속도 완화 기대.
