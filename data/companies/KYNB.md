# Kyntra Bio, Inc. (KYNB)

## 회사 개요
2026년 1월 7일 FibroGen이 사명을 Kyntra Bio로 변경했고, 1월 8일 개장부터 Nasdaq 티커가 KYNB로 거래 개시됐다. 종양·희귀질환 중심으로 사업 축을 재편했다. 핵심 자산은 CD46 표적 항체-약물 접합체(ADC) FG-3246(전립선암)과 저위험 골수형성이상증후군(LR-MDS) 빈혈 대상 HIF-PHI 경구제 roxadustat다. FG-3180은 CD46 표적 PET 영상 바이오마커(동반 진단)다. 2025년 FibroGen China 매각·선순위 담보 텀론 상환으로 캐시런웨이를 2028년까지 연장했다.

## 카탈리스트
- FG-3246 + enzalutamide 병용 Phase 1b/2(연구자 주도, mCRPC): ASCO GU 2026(2026-02-26~28)에서 데이터 발표 완료 — 긍정적 결과.
- FG-3246 단독요법 Phase 2(용량 최적화, FG-3180 PET 동반): 2026년 하반기 중간결과 예상.
- roxadustat LR-MDS 빈혈: FDA에 pivotal Phase 3 프로토콜 제출, Phase 3 개시 준비. FDA가 MDS 대상 Orphan Drug 지정 부여.

## 적응증
- FG-3246 · 전이성 거세저항성 전립선암(mCRPC) · Phase 2 단독요법(용량 최적화) / Phase 1b/2 enzalutamide 병용 · CD46 표적 ADC(MMAE payload)
- FG-3180 · 전립선암(CD46 PET 영상 동반진단) · Phase 2 · CD46 표적 PET 영상 바이오마커
- roxadustat · 저위험 MDS 빈혈 · Phase 3 준비 · HIF-PHI 경구제(유럽·일본 등서 CKD 빈혈 승인됨)

## 임상 디자인
FG-3246 + enzalutamide Phase 1b/2 (NCT05011188, 연구자 주도):
- N=44 바이오마커 비선별 mCRPC(용량 증량 17 + 확장 27), 사전 ARPI ≥1회 진행 필수, 60%+ 가 ARPI 2종 이상 경험.
- 권장 Phase 2 용량 2.1 mg/kg + enzalutamide 160 mg/day. Phase 2는 복합반응(PSA50 및/또는 RECIST ORR) 평가.
- 전체 코호트: 중앙 rPFS 7.0개월, 복합반응률 21%.
- 사전 ARPI 1회군: 중앙 rPFS 10.1개월, 반응률 40%.
- FG-3180 PET 흡수도 높은 환자에서 PSA50 반응 상승 경향(p=0.053).

## Modality / MOA
- FG-3246: 완전인간 항체-약물 접합체(ADC). CD46(세포 표면 수용체) 결합 시 내재화 유도, MMAE(항유사분열) payload 전달.
- roxadustat: 저산소유도인자 프롤릴수산화효소 억제제(HIF-PHI) 경구제.

## 파트너
- roxadustat: 레거시 협력(AstraZeneca / Astellas 지역별 라이선스) — 구체 거래조건은 본 패스 출처에서 미확인.

## 메모
- 2025년 FibroGen China 매각 + 선순위 담보 텀론(Morgan Stanley Tactical Value 시설) 상환 완료 → 캐시런웨이 2028년까지 연장. 정확한 현금 잔액 수치는 본 패스 출처에 비공개.
- 사명 변경은 사업 재편(종양·희귀질환 집중)의 일환. 이전 FibroGen 시절 핵심이던 roxadustat CKD 사업은 China 매각으로 축소.
- 마이크로캡(screener 유래). 임상 단계, 상업 매출 없음.
