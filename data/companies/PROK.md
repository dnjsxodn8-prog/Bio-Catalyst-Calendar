# ProKidney (PROK)

## 카탈리스트
REACT (신장 자가세포 치료) Phase 2b REGEN-006 최종 데이터 발표 (2026년 하반기-2027년 예상)

## 회사 개요
ProKidney는 만성 신장 질환(CKD) 특화 세포치료 바이오텍. Winston-Salem, NC 기반. 핵심 기술: REACT(Renal Autologous Cell Therapy) — 환자 자신의 신장 세포를 추출·배양해 신장에 재주입함으로써 기능 보존 목표.

CKD는 전 세계 약 8억 명, 투석 시장 연 $90B. REACT는 투석 지연 또는 회피를 목표로 하는 첫 세포치료 시도.

## 임상 디자인
- 디자인: Randomized, controlled (REGEN-006 Phase 2b)
- Arms: REACT vs 표준치료 대조
- N: ~200명
- Phase: Phase 2b
- Primary endpoint: GFR 감소율 기저 대비 변화 (12-24개월)
- Secondary endpoint: 투석 도달까지 기간, 안전성

## 타겟 질환
**만성 신장 질환 (Chronic Kidney Disease, CKD) — 당뇨성 신병증**

GFR 단계적 저하. 말기 신부전(ESRD)으로 진행 시 투석 또는 이식 필요.

- 환자 수: 미국 ~3,700만 명 CKD, 이 중 당뇨성 신병증 ~40%
- 병리: 사구체 경화, 세뇨관 손실 → GFR 저하 → 노폐물 배출 불가

## 기존 치료제
SGLT2i (Jardiance, Farxiga), RAS 차단제, Kerendia. 모두 진행 억제제지 회복 불가. REACT는 세포 보충으로 기능 회복 시도.

## 사전 공개 임상
- **REGEN-001 Phase 2a · NCT02565459 · CKD**
  결과: Positive (2022) — 12개월 GFR 감소율 유의미한 억제, 안전성 양호
- **REGEN-006 Phase 2b · NCT04977570 · 진행 중**
  결과: 최종 데이터 2026-2027년 예상

## Modality
**REACT (Renal Autologous Cell Therapy)**
자가 신장 선택적 세포 치료 (type: Cell Therapy)

신장 생검 → Neo Kidney Augment(NKA) 세포 선별 (PTEC, 세뇨관 세포 포함) → 배양 증폭 → 신장 피질 내 직접 주입.

## MOA
- 타겟: 손상된 신장 세뇨관 및 사구체 세포
- 기전: 자가 신장 세포(PTEC, 집합관 세포 등) 보충 → 잔존 네프론 기능 지지 및 재생 촉진. Paracrine 효과(성장인자, 항염 사이토카인 분비) → 섬유화 억제.
- 병태생리: CKD에서 세뇨관 세포 소실 → GFR 진행성 감소. REACT가 기능 세포 보충해 GFR 감소 속도 완화.
