# Milestone Pharmaceuticals (MIST)

## 카탈리스트
Cardamyst (etripamide) 발작성 상심실성 빈맥(PSVT) 미국 시판 허가 및 출시 준비

## 회사 개요
캐나다/미국 기반 심장 부정맥 전문 바이오텍. Cardamyst(etripamide, 비강 내 verapamil 유사 칼슘채널차단제)는 PSVT 급성 종료 목적 비강 흡입 제제. 2024년 FDA Complete Response Letter(CRL) 수령 후 재심사 제출.

소형 심장 희귀 적응증 전문 회사. 단일 파이프라인 집중 리스크.

## 임상 디자인
- 시험명: NODE-301 (etripamide PSVT)
- NCT: NCT04462393
- Phase: Phase 3
- N: ~431명 (PSVT 발작 시 치료)
- 디자인: Randomized, double-blind, placebo-controlled
- Arms: etripamide 70mg 비강 스프레이 vs 위약 (발작 발생 시 투여)
- Primary endpoint: 발작 종료(NSR 전환) 후 30분 이내 지속 유지율
- Secondary endpoint: 발작 종료 시간 중앙값, 응급실 방문률 감소

## 타겟 질환
**발작성 상심실성 빈맥 (Paroxysmal Supraventricular Tachycardia, PSVT)**

방실결절 재진입 또는 보조전도로 매개 갑작스러운 빠른 심박수(>150/분). 삽화적 발작으로 삶의 질 저하.

- 환자 수: 미국 PSVT ~570만 명, 연간 응급 방문 ~500,000건
- 병리: 방실결절 재진입(AVNRT) 또는 보조전도로(WPW) → 자동성 또는 방아쇠 활동으로 빈맥 발생. 칼슘채널차단제로 방실결절 전도 차단 → 빈맥 종료.

## 기존 치료제
정맥 adenosine/verapamil (응급실 한정). 미주신경 자극(Valsalva 등). Cardamyst는 최초 비강 자가 투여 제제 — 응급실 방문 없이 가정에서 치료 가능.

## 사전 공개 임상
- **NODE-301 Phase 3 · NCT04462393**
  결과 (2023): Primary endpoint 미달 — etripamide 63.5% vs 위약 51.3% (NSR 전환, p=0.052, 통계적 유의성 아슬). 사전 지정 AVNRT 서브그룹에서 유의한 효과. → FDA 2024년 CRL 발행 (추가 데이터 요구).
- **NODE-202 Phase 2 · NCT03464383**
  결과 (2019): PSVT 종료 효과 확인, 안전성 프로파일 양호. Phase 3 근거.

## Modality
**Etripamide (Cardamyst)**
Intranasal calcium channel blocker (phenylalkylamine, verapamil analog) (type: Small Molecule)

비강 내 파우더 흡입. 빠른 흡수로 혈중 농도 신속 도달. Verapamil 유사 구조 — 방실결절 칼슘채널 차단.

## MOA
- 타겟: L-type 전압 의존성 칼슘채널 (CaV1.2/1.3) — 방실결절
- 기전: 비강 점막 흡수 → 빠른 혈중 Cmax → 방실결절 L-type 칼슘채널 차단 → 방실 전도 속도 감소 + 불응기 연장 → 재진입 회로 차단 → NSR 전환.
- 병태생리: AVNRT에서 방실결절의 느린·빠른 전도로 재진입. 칼슘채널 차단으로 느린 전도로 기능 억제 → 재진입 회로 소멸.
