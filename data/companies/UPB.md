# Upstream Bio (UPB)

## 카탈리스트

- Phase 3 severe asthma 첫 환자 투여 — 2027 Q1 예정 (verekitug 400mg q12w)
- Phase 3 CRSwNP 첫 환자 투여 — 2027 Q1 예정 (동일 용량 전략)
- VENTURE (COPD) Phase 2 중간 데이터 — 2026 하반기 예상 (60%+ 모집 완료)
- VIBRANT (CRSwNP) 추가 분석 — 2026 AAAAI 발표 완료 (2026-03)
- VALIANT (severe asthma) Positive topline — 2026-02-11 발표 완료

## 회사 개요

현재 임상 중 유일한 TSLPR 길항 단클론항체 verekitug로 중증 천식·CRSwNP Phase 2를 모두 Positive로 마친 단일자산 집중형 임상 바이오텍이다. 2027 Q1 Phase 3 첫 환자 투여가 다음 카탈리스트로, 현금 $341M으로 진입은 확정이나 최종 데이터까지 추가 자금조달이 필요할 가능성이 높다.

## 임상 디자인

**VALIANT (Severe Asthma, Phase 2, 완료)**
- 디자인: Global, randomized, double-blind, placebo-controlled, dose-ranging
- Arms: verekitug 100mg q12w / 100mg q24w / 400mg q24w / placebo (SC)
- N: 478명
- Primary: AAER(연간 천식 악화율) 감소
- 결과: Positive(2026-02-11) — 100mg q12w에서 AAER 56% 감소(p<0.0003), 400mg q24w 39% 감소(p<0.02). 발표 후 주가 -47%(기대치 하회 해석이나 1차 endpoint 충족).

**VIBRANT (CRSwNP, Phase 2, 완료)**
- 디자인: Global, randomized, double-blind, placebo-controlled
- Arms: verekitug 100mg q12w / placebo (SC)
- N: 81명, 24주
- Primary: 내시경 비용종 점수(NPS) 기저 대비 변화
- 결과: Positive(2025-09-02) — NPS -1.8(placebo-adjusted, p<0.0001), 수술/스테로이드 필요 76% 감소

## 타겟 질환

**중증 천식(Severe Asthma)**: 미국 약 250만 명(전체 천식의 5~10%). 알러젠·바이러스 → 기도 상피 TSLP 분비 → ILC2 활성화 → Th2 사이토카인(IL-4/5/13) 폭포 → 기도 호산구 침윤·점막 과증식. 고용량 흡입 스테로이드에도 반응 불충분.

**CRSwNP**: 미국 약 300만 명. 비강 점막 Th2 염증 → IL-5 매개 호산구 침윤 → 비용종 형성. TSLP가 상류 신호로 Dupilumab(IL-4Rα) 등 하류 타겟과 차별화.

## 기존 치료제

- **중증 천식**: Dupixent(IL-4Rα), Fasenra(IL-5), Nucala(IL-5), Tezspire(TSLP 리간드 차단), 흡입 스테로이드/LABA
- **CRSwNP**: Dupixent(FDA 승인), Xolair(omalizumab), 전신 스테로이드, 수술
- Tezspire는 TSLP 리간드 차단(q4w), verekitug는 TSLPR 수용체 차단(q12w) — 동일 경로 다른 타겟·투여 주기

## 사전 공개 임상

- **Phase 1b MAD(천식)**: 전 용량에서 TSLPR 완전 점유(2주 달성, 24주 유지). FeNO -54%, 호산구 -65%, IL-5 -64%. 내약성 양호. (Singh et al., Clin Pharmacol Ther 2026)
- **VIBRANT NCT06164704 Phase 2**: Positive(2025-09-02) — NPS -1.8(p<0.0001), NCS -0.8(p=0.0003), SAE 없음.
- **VALIANT NCT06196879 Phase 2**: Positive(2026-02-11) — AAER 56% 감소(100mg q12w, p<0.0003). FEV₁·FeNO 통계적 유의 개선.

## Modality

**verekitug(UPB-101)**
Recombinant fully human IgG1 mAb, TSLPR 길항제. 피하주사, q12w 또는 q24w. Astellas에서 인수. 현재 임상 중 유일한 TSLPR 직접 차단 항체. Dupilumab(IL-4Rα)·Tezspire(TSLP 리간드)와 달리 수용체 자체 차단.

## MOA

- 타겟: TSLPR(thymic stromal lymphopoietin receptor, CRLF2/IL-7Rα 이형이합체)
- 기전: verekitug가 TSLPR 리간드 결합 도메인에 고친화도 결합 → TSLP의 수용체 결합 경쟁적 차단 + 수용체 이형이합체 복합체 선점 → JAK1/JAK2-STAT5 신호 전달 억제
- 병태생리: 알러젠·바이러스·오염물질 → 기도/비강 상피 손상 → TSLP 분비(염증 폭포 최상류) → ILC2 활성화·Th2 편향·IL-4/5/13/33 상승 → 호산구 침윤·점막 리모델링. TSLPR 차단으로 하류(IL-4Rα, IL-5) 차단 대비 더 넓은 사이토카인 스펙트럼 억제.
