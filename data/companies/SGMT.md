# Sagimet Biosciences (SGMT)

## 카탈리스트
Denifanstat (TVB-2640) Phase 2 F4 MASH 개시 (2026H2); 미국 Phase 3 여드름 개시; TVB-3567 Phase 2 개시 (2026H2)

## 회사 개요
Sagimet Biosciences는 FASN(지방산 합성효소) 억제제 플랫폼 기반 기업. 선도 물질 Denifanstat(TVB-2640)은 중국 중등~중증 여드름 Phase 3 성공, NMPA NDA 허가 신청 수리(2025년 12월). 미국 여드름 Phase 3 준비 중. MASH F4에서 resmetirom 병용 Phase 2 POC 개시 예정(2026H2). $175M 공모로 자금 확보.

## 타겟 질환
**MASH (대사기능장애 관련 지방간염, F4 단계)**

지방 과축적 + 염증 + 섬유화 → 간경변. FASN 억제로 de novo lipogenesis 차단.

- 환자 수: 미국 약 1,500만~2,000만 명 MASLD, 중증(F4) 비율 높음
- 병리: 과잉 지방산 합성 → 간세포 손상·염증·섬유화

## 기존 치료제
Resmetirom(Rezdiffra, Madrigal, FDA 승인 2024). Denifanstat은 resmetirom과 상보적 병용 가능성.

## 사전 공개 임상
- **Denifanstat 중국 여드름 Phase 3 · 양성 (2025)**
  결과: Positive. NMPA NDA 수리(2025.12).
- **Denifanstat + Resmetirom Phase 1 PK · 완료**
  결과: 안전 신호 없음. F4 MASH Phase 2 개시 계획.

## Modality
**Denifanstat (TVB-2640)**
Oral FASN (fatty acid synthase) inhibitor (type: Small Molecule)

경구 1일 1회. FASN을 비가역적으로 억제 → 지방산 신합성 차단.

## MOA
- 타겟: FASN (Fatty Acid Synthase)
- 기전: FASN의 β-ketoacyl reductase 도메인 억제 → palmitate 등 지방산 신합성 차단 → 간 지질 축적 감소, 종양세포 지질 의존성 억제
- 병태생리: MASH에서 과잉 DNL이 간세포 손상 유발. 여드름에서 피지선 FASN 과발현. 종양에서 FASN 업레귤레이션.
