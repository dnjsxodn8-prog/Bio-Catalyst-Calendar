# SAB Biotherapeutics (SABS)

## 카탈리스트
SAB-142 SAFEGUARD Phase 2b (Stage 3 T1D) — 등록 완료 2026년 말 목표, topline 데이터 2027 하반기 예상

## 회사 개요
형질전환 소(transgenic cattle) 기반 완전 인간 폴리클로날 항체 생산 플랫폼 보유 임상 단계 바이오텍. 리드 에셋 SAB-142는 Stage 3 자가면역 1형 당뇨병(T1D)을 표적으로 하는 폴리클로날 생물학제제. 2025-07 Sanofi 전략적 투자 포함 $175M 사모펀딩 성공으로 SAFEGUARD Phase 2b 전액 자금 확보. Phase 1 긍정적 결과(혈청병 없음, 저면역원성, 일시적 림프구감소증 on-target). 2026 포커스: SAFEGUARD 등록 완료.

## 매출
없음 (임상 단계). 2025-07 $175M 조달(Sanofi 참여).

## 임상 디자인
- 시험명: SAFEGUARD (Phase 2b, 잠재적 등록 임상)
- 디자인: 무작위 배정, 이중 맹검, 위약 대조 (글로벌 다기관)
- 투여: SAB-142 vs. 위약 (만성 외래 투여)
- 지역: 미국, 호주, 뉴질랜드, 유럽(예정)
- Phase: Phase 2b (잠재적 등록)
- Primary endpoint: C-peptide 보존 (2시간 혼합식 부하 검사 AUC)
- Secondary endpoint: HbA1c, 인슐린 용량, 저혈당 빈도, 안전성

## 타겟 질환
**Stage 3 자가면역 1형 당뇨병 (T1D, Type 1 Diabetes — Stage 3)**

자가면역 반응으로 췌장 β세포가 파괴되어 인슐린 분비 불능. Stage 3 = 고혈당 증상 발생 시점. 기존 teplizumab(CD3 항체, tzield)이 Stage 2 → Stage 3 진행 지연 승인. SAB-142는 Stage 3 환자에서 자가면역 공격 조절로 잔여 β세포 보존 목표.

- 환자 수: 미국 신규 T1D 연간 ~64,000명 (소아·청소년 포함)
- 병리: T세포·B세포 매개 자가면역 → 항 GAD, IA-2 자가항체 생성 → β세포 파괴

## 기존 치료제
Tzield (teplizumab, Sanofi/Provention Bio) — anti-CD3, Stage 2 T1D 진행 지연 유일 승인. 인슐린(대체 요법). SAB-142는 폴리클로날 항체로 복수 에피토프 표적 — 단일 항체 대비 폭넓은 면역 조절 기대.

## 사전 공개 임상
- **Phase 1 (건강인·성인 T1D) — 2025-12 결과 발표**
  결과: 혈청병 없음 (단백질 기반 치료에서 중요). 전 코호트(건강인, 재투여, T1D)에서 저면역원성 확인. 일시적 림프구감소증 on-target 바이오마커 확인, 1~3일 내 기저치 회복. 만성 외래 투여 가능성 확인.
- **SAFEGUARD (Phase 2b)**: 첫 환자 투여 완료. 2026년 말 등록 완료 목표. Topline 2027H2.

## Modality
**SAB-142**
완전 인간 폴리클로날 IgG 항체 혼합물 (유형: Biologic)

형질전환 소(인간 면역글로불린 유전자 보유)에서 생산. 복수 자가면역 에피토프(T세포·B세포 활성화 경로)를 동시 표적. 단일클로날 항체 대비 더 넓은 면역 조절 범위 기대. 반복 투여 시 혈청병·면역원성 최소화 확인.

## MOA
- 타겟: 다중 면역 세포 에피토프 (T세포, B세포 활성화 경로)
- 기전: 폴리클로날 인간 항체가 자가면역 공격에 관여하는 복수 면역 세포 표적 동시 조절 → T1D에서 자가반응성 면역 세포 활성 억제 → 잔여 β세포 보존 → 내인성 인슐린 분비 유지
- 병태생리: Stage 3 T1D — β세포 자가면역 파괴 진행 중. SAB-142의 폭넓은 면역 조절로 파괴 속도 지연 시도.
