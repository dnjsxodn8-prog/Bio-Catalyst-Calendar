# Viking Therapeutics (VKTX)

## 카탈리스트
- VANQUISH-1 (VK2735 SC, 비만): Phase 3 등록 완료 2025.11. 1차 결과 2026H2 예상 (78주 투여, primary endpoint 도달 시점)
- VANQUISH-2 (VK2735 SC, T2D+비만): Phase 3 등록 완료 2025.12. 결과 2026-2027 예상
- VK2735 Oral Phase 3 시작: VENTURE-Oral Phase 2 양호(2025.8) → Phase 3 설계 진행 중
- VK2809 (MASH) Phase 3 계획: VOYAGE Phase 2b 52주 histology 성공 → Phase 3 IND/파트너링 논의 중

## 회사 개요
GLP-1/GIP 이중작용제 VK2735(SC + 경구)로 비만 시장 진입을 노리는 임상 단계 바이오텍으로, 현금 $706M 보유 및 파트너 없이 독자 개발 중. 2026년 하반기 VANQUISH-1 Phase 3 결과가 핵심 촉매로, 성공 시 SC·경구 이중 포뮬레이션으로 Zepbound와 직접 경쟁하는 포지션을 확보한다.

## 임상 디자인
VANQUISH-1 (VK2735 SC, 비만)
- 디자인: Randomized, double-blind, placebo-controlled, multicenter
- Arms: VK2735 7.5 mg SC QW / 12.5 mg SC QW / 17.5 mg SC QW / Placebo
- N: ~4,500명 (BMI ≥30, 또는 BMI ≥27 + 동반질환 1개 이상)
- Phase: Phase 3 (VANQUISH-1)
- Primary endpoint: 체중 기저 대비 % 변화 (78주)
- Secondary endpoint: ≥5%/≥10%/≥15%/≥20% 체중 감량 달성률, 허리둘레, 대사 지표

VANQUISH-2 (VK2735 SC, T2D + 비만)
- 디자인: Randomized, double-blind, placebo-controlled, multicenter
- Arms: VK2735 7.5 mg / 12.5 mg / 17.5 mg SC QW / Placebo
- N: ~1,100명 (T2D + 비만/과체중)
- Phase: Phase 3
- Primary endpoint: 체중 기저 대비 % 변화 (78주)
- Secondary endpoint: HbA1c 변화, ≥5%/≥10% 체중 감량 달성률

VENTURE-Oral (VK2735 경구, 비만)
- 디자인: Randomized, double-blind, placebo-controlled, multicenter
- Arms: 6개 용량군 (증량 방식) / Placebo
- N: 280명 (BMI ≥30, 또는 BMI ≥27 + 동반질환)
- Phase: Phase 2 (완료)
- Primary endpoint: 체중 기저 대비 % 변화 (13주)
- Secondary endpoint: 안전성·내약성, PK

## 타겟 질환
비만 (Obesity)
- 환자 수: 미국 성인 ~1억 명 (BMI ≥30), 과체중 포함 시 ~2억 명
- 병리: 에너지 항상성 조절 이상. 시상하부 GLP-1/GIP 신호 감소 → 식욕 과항진 + 지방 축적. 인슐린 저항성, T2D, 심혈관 위험 상승.

MASH (Metabolic Dysfunction-Associated Steatohepatitis)
- 환자 수: 미국 ~1,500만 명 (MASH + 섬유화 동반)
- 병리: 간내 지방 과다 축적 → 염증·세포사 → 섬유화 진행. THR-β 활성화로 간 선택적 지방산 산화 촉진, 섬유화 억제.

## 기존 치료제
비만: Wegovy (NVO, GLP-1), Zepbound (LLY, GLP-1/GIP), Saxenda (NVO). VK2735는 Zepbound과 같은 dual agonist 계열.

MASH: Rezdiffra (resmetirom, Madrigal, THR-β, 2024 FDA 승인 첫 MASH 치료제). VK2809은 같은 THR-β 타겟. Ocaliva (obeticholic acid, FXR) 시장 퇴장.

## 사전 공개 임상
- VENTURE SC (VK2735 SC) · NCT06016296 · Phase 2  결과: Positive (2024.2) — 13주 최대 14.7% 체중 감량 vs placebo -1.7%. 88%에서 ≥10% 감량. Primary/Secondary 전 달성.

- VENTURE-Oral (VK2735 oral) · NCT06068946 · Phase 2  결과: Positive (2025.8) — 13주 최대 12.2% 체중 감량 (vs placebo 대비 최대 10.9% 차이). 안전성 良好. "Mixed"는 일부 용량 비교군 간 격차 때문이나 primary endpoint 달성.

- VOYAGE (VK2809 MASH) · NCT04173065 · Phase 2b  결과: Positive 52주 histology (2024.5) — MASH resolution 69% vs placebo 29%. 섬유화 개선 51% vs 34%. 간지방 감소 중앙값 38-55% (12주). Primary + histology secondary 모두 달성.

- VK0214 X-ALD · Phase 1b  결과: Positive (2024.10) — 28일 투여 안전성 확인. 혈장 VLCFA 유의미한 감소. ABCD2 경로 활성화 확인.

## Modality
VK2735 (SC / Oral)- SC 제형: Peptide-based GLP-1/GIP dual receptor agonist. 주 1회 피하주사.
- Oral 제형: 소분자 경구 GLP-1/GIP dual agonist. 1일 1회 정제. 식사 독립적 흡수 목표.

VK2809Oral small molecule. Selective thyroid hormone receptor beta (THR-β) agonist. 간 선택적 작용.

VK0214Oral small molecule. Selective THR-β agonist. X-ALD 대상 ABCD2 유도.

## MOA
VK2735 (비만)
- 타겟: GLP-1R (glucagon-like peptide-1 receptor) + GIPR (glucose-dependent insulinotropic polypeptide receptor)
- 기전: GLP-1R 활성화 → 췌장 β세포 glucose-dependent 인슐린 분비 ↑, glucagon 억제, 위배출 지연, 시상하부 식욕 억제. GIPR 활성화 → 인슐린 감수성 개선, 추가 체중 감량 시너지. 단일 GLP-1R 대비 dual agonism으로 체중 감량 효과 강화.
- 병태생리: GLP-1/GIP 신호 부족으로 에너지 항상성 이상 → 식욕 과항진, 지방 축적. Dual agonism이 두 경로 동시 교정.

VK2809 (MASH)
- 타겟: THR-β (thyroid hormone receptor beta)
- 기전: 간 선택적 THR-β 활성화 → 미토콘드리아 지방산 β-산화 ↑, 간 지방 생성 억제, LDL-C 감소. 간 선택성으로 심장·뼈 등 말초 THR-α 부작용 최소화.
- 병태생리: MASH는 간 내 지방 과다 → 산화 스트레스 → 염증·세포사 → 섬유화. THR-β 활성화로 지방산 대사 정상화.
