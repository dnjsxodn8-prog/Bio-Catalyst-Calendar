# Viking Therapeutics (VKTX)

## 카탈리스트
- VANQUISH-1 (VK2735 SC, 비만): Phase 3 등록 완료 2025.11. 1차 결과 2026H2 예상 (78주 투여, primary endpoint 도달 시점)
- VANQUISH-2 (VK2735 SC, T2D+비만): Phase 3 등록 완료 2025.12. 결과 2026-2027 예상
- VK2735 Oral Phase 3 시작: VENTURE-Oral Phase 2 양호(2025.8) → Phase 3 설계 진행 중
- VK2809 (MASH) Phase 3 계획: VOYAGE Phase 2b 52주 histology 성공 → Phase 3 IND/파트너링 논의 중

## 회사 개요
GLP-1/GIP 이중작용제 VK2735(SC + 경구)로 비만 시장 진입을 노리는 임상 단계 바이오텍으로, 현금 $706M 보유 및 파트너 없이 독자 개발 중. 2026년 하반기 VANQUISH-1 Phase 3 결과가 핵심 촉매로, 성공 시 SC·경구 이중 포뮬레이션으로 Zepbound와 직접 경쟁하는 포지션을 확보한다.

## 매출
상업화 제품 없음, 매출 N/A. 현금 ~$706M (2025년 말), 2027년 이후까지 자금 충분.

## 플랫폼
- **VK2735 SC**: GLP-1/GIP dual agonist, 주 1회 피하주사 (Peptide)
- **VK2735 Oral**: 경구 GLP-1/GIP dual agonist, 1일 1회 정제 (Small Molecule)
- **VK2809**: Selective THR-β agonist, 간 선택적 경구 소분자 (MASH)
- **VK0214**: Selective THR-β agonist, X-ALD 대상 ABCD2 유도 (Rare Disease)

**VENTURE-Oral (VK2735 경구, 비만, Phase 2 완료)**
- 디자인: Randomized, double-blind, placebo-controlled, multicenter

## 타겟 질환
**비만 (Obesity)**
- 환자 수: 미국 성인 ~1억 명 (BMI ≥30), 과체중 포함 시 ~2억 명
- 병리: 에너지 항상성 조절 이상. 시상하부 GLP-1/GIP 신호 감소 → 식욕 과항진 + 지방 축적. 인슐린 저항성, T2D, 심혈관 위험 상승.

**MASH (Metabolic Dysfunction-Associated Steatohepatitis)**
- 환자 수: 미국 ~1,500만 명 (MASH + 섬유화 동반)
- 병리: 간내 지방 과다 축적 → 염증·세포사 → 섬유화 진행. THR-β 활성화로 간 선택적 지방산 산화 촉진, 섬유화 억제.

## 기존 치료제
- Wegovy (NVO, GLP-1): 비만 1선 치료
- Zepbound (LLY, GLP-1/GIP dual agonist): VK2735와 동일 계열, 직접 경쟁
- Saxenda (NVO, GLP-1): 비만 치료제
- Rezdiffra (resmetirom, Madrigal, THR-β): 2024년 FDA 승인 첫 MASH 치료제. VK2809과 동일 타겟.

## 사전 공개 임상
- **VENTURE SC (VK2735 SC) NCT06016296 Phase 2**: Positive (2024.2) — 13주 최대 14.7% 체중 감량 vs placebo -1.7%. 88%에서 ≥10% 감량.
- **VENTURE-Oral (VK2735 oral) NCT06068946 Phase 2**: Positive (2025.8) — 13주 최대 12.2% 체중 감량 (vs placebo 대비 최대 10.9% 차이). Primary endpoint 달성.
- **VOYAGE (VK2809 MASH) NCT04173065 Phase 2b**: Positive 52주 histology (2024.5) — MASH resolution 69% vs placebo 29%. 섬유화 개선 51% vs 34%.
- **VK0214 X-ALD Phase 1b**: Positive (2024.10) — 28일 안전성 확인, 혈장 VLCFA 유의미한 감소.

## Modality
**VK2735 (SC / Oral)**
SC 제형: Peptide-based GLP-1/GIP dual receptor agonist, 주 1회 피하주사. Oral 제형: 소분자 경구 GLP-1/GIP dual agonist, 1일 1회 정제.

## MOA
- 타겟: GLP-1R (glucagon-like peptide-1 receptor) + GIPR (glucose-dependent insulinotropic polypeptide receptor)
- 기전: GLP-1R 활성화 → 인슐린 분비 ↑, glucagon 억제, 위배출 지연, 식욕 억제. GIPR 활성화 → 인슐린 감수성 개선, 추가 체중 감량 시너지.
- 병태생리: GLP-1/GIP 신호 부족으로 에너지 항상성 이상 → 식욕 과항진, 지방 축적. Dual agonism이 두 경로 동시 교정.
