# Gyre Therapeutics (GYRE)

## 카탈리스트
Hydronidone (F573) Phase 3 중국 NDA 승인 결과 / 미국 IND 신청 (2026)

## 회사 개요
중국 기반 간섬유화·NASH 특화 소형 바이오텍. 리드 에셋 hydronidone(F573)는 TGF-β/smad 경로 억제 소분자로 중국에서 대규모 Phase 3 완료. NDA 제출 후 승인 검토 중. 미국 시장 진출도 계획.

Hutchison 계열사에서 분사. 중국 임상 데이터 기반 글로벌 확장 전략.

## 임상 디자인
- 시험명: Hydronidone Phase 3 (중국)
- NCT: NCT04064359
- Phase: Phase 3
- N: 약 600명
- 대상: 만성 B형 간염 동반 간섬유화 (Ishak F2-F4)
- 투여: Hydronidone 200mg 1일 3회 vs placebo, 72주
- Primary endpoint: Ishak 섬유화 점수 1단계 이상 개선 + 악화 없음
- 디자인: Randomized, double-blind, placebo-controlled

## 타겟 질환
**간섬유화 / NASH (Hepatic Fibrosis / Non-Alcoholic Steatohepatitis)**

만성 간 염증·손상으로 콜라겐 과잉 축적 → 간경변·간암 진행.

- 환자 수: 중국 만성 B형 간염 환자 약 8,600만 명 (섬유화 위험군 다수), 미국 NASH 환자 약 1,500만 명
- 병리: TGF-β/smad 신호 과활성 → 간성상세포(HSC) 활성화 → 콜라겐 분비

## 기존 치료제
중국: 근본 항섬유화 치료제 없음 (항바이러스제 표준). 미국: resmetirom (Rezdiffra, NASH 최초 승인, 2024). Hydronidone은 TGF-β 경로 직접 차단.

## 사전 공개 임상
- **Hydronidone Phase 2 (중국) · 정보 미입력 · Phase 2**
  결과: Positive - 간섬유화 개선 신호 확인, 안전성 양호. Phase 3 진입 근거.
- **F573 Phase 1 · 정보 미입력 · Phase 1**
  결과: 안전성·PK 확인

## Modality
**Hydronidone (F573)**
TGF-β/Smad pathway inhibitor (type: Small Molecule)

경구 소분자. TGF-β 신호 억제로 간성상세포 활성화 차단 → 콜라겐 합성 감소.

## MOA
- 타겟: TGF-β/Smad2/3 신호 경로
- 기전: TGF-β 수용체 신호 억제 → Smad2/3 인산화 차단 → 간성상세포(HSC) 활성화·증식 억제 → 콜라겐 I형/III형 합성 감소 → 섬유화 역전
- 병태생리: 만성 간 손상(HBV, 알코올, 지방) → TGF-β 과발현 → HSC 근섬유모세포 전환 → ECM 과적. Hydronidone이 이 핵심 경로 차단.
