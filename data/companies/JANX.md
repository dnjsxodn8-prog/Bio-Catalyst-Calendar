# Janux Therapeutics (JANX)

## 카탈리스트
JANX007 (PSMA x CD3 T세포 인게이저) Phase 1 용량 확장 코호트 데이터 — 2026년 ASCO/ESMO 발표 예정; JANX008 (EGFR x CD3) Phase 1 초기 데이터

## 회사 개요
Janux Therapeutics는 독자 TRACTr(Tumor-activated T Cell Engager) 플랫폼 기반 차세대 이중특이항체 개발 전문 임상 단계 바이오텍. 기존 T세포 인게이저의 사이토카인 방출 증후군(CRS) 한계를 극복하기 위해 종양 미세환경에서만 활성화되는 마스킹(masking) 기술 적용. 전립선암(JANX007, PSMA 타겟)과 고형암(JANX008, EGFR 타겟) 임상 진행 중.

## 매출
상업화 제품 없음. 비수익 단계.

## 플랫폼
- TRACTr (Tumor-activated T Cell Engager) 플랫폼
- 종양 특이적 활성화 이중특이항체 (Masked bispecific)
- PSMA, EGFR, HER2 등 다수 타겟 적용 가능

## 적응증
Oncology (전립선암, 고형암)

- 없음 (독자 개발)

비수익 단계.

## 임상 디자인
- 디자인: Open-label, dose-escalation (3+3 design)
- Arms: JANX007 단독 투여 (용량 확장 코호트)
- N: ~60–80명 (전체 Phase 1)
- Phase: Phase 1 (용량 확장 단계)
- Primary endpoint: 안전성, MTD/RP2D 결정, DLT; 보조: PSA 반응률, 객관적 반응률(ORR)
- NCT: NCT05476497

## 타겟 질환
**전이성 거세저항성 전립선암 (mCRPC)**

안드로겐 차단 요법 실패 후 진행한 전립선암.

- 환자 수: 미국 연간 신규 ~35,000–40,000명 (mCRPC 단계)
- 병리: AR 신호 지속 활성화 + 안드로겐 독립적 생존 → 항호르몬 요법 내성

## 기존 치료제
Enzalutamide, Abiraterone, Docetaxel, 177Lu-PSMA-617(Pluvicto). T세포 인게이저는 mCRPC에서 미승인 (임상 단계 경쟁자: AMG 160, CC-1 등).

## 사전 공개 임상
- **JANX007 Phase 1 용량 탐색 · NCT05476497**
  초기 결과 (2024 ASCO): CRS 발생 최소화 확인, PSA 감소 반응 관찰 (용량 의존적). 마스킹 기전 임상적 검증.
- **JANX008 Phase 1 (EGFR+ 고형암) · NCT06040541**
  2024년 개시. 초기 안전성 데이터 수집 중.

## Modality
**JANX007**
PSMA x CD3 TRACTr 이중특이항체 (type: Antibody)

정맥주사(IV). 종양 미세환경 내 프로테아제가 마스크 도메인 절단 → 활성화 → T세포-종양세포 면역시냅스 형성.

## MOA
- 타겟: PSMA (전립선암 세포), CD3ε (T세포)
- 기전: 마스킹 펩타이드가 순환혈액 중 항체 활성 억제 → 종양 미세환경 내 MMP/uPA 프로테아제 절단 → PSMA·CD3 동시 결합 활성화 → T세포 직접 모집 → 퍼포린/그랜자임 매개 종양세포 사멸
- 병태생리: mCRPC에서 PSMA 과발현 (80%+). 기존 T세포 인게이저는 전신 CD3 활성화로 CRS 제한 → TRACTr 마스킹으로 전신 독성 회피

## 메모
TRACTr 마스킹 플랫폼이 핵심 차별점. CRS 없이 고용량 투여 가능성이 관건. 2026년 Phase 1 용량 확장 코호트 반응률 데이터가 밸류에이션 변곡점.
