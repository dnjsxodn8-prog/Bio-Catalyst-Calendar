# Candel Therapeutics (CADL)

## 카탈리스트
CAN-2409 전립선암 BLA 제출 — 2026 Q4 (RMAT 지정으로 우선 심사 경로)

## 회사 개요
복제 불가 아데노바이러스 기반 유전자 치료제 CAN-2409(아데노바이러스-HSV-TK) 개발 임상 기업. 중·고위험 국소 전립선암 Phase 3에서 DFS 30% 개선(HR 0.70, p=0.016) 및 전립선암 특이적 DFS 38% 개선(HR 0.62, p=0.005) 달성. FDA RMAT 지정으로 우선 심사 + 롤링 BLA 제출 가능.

Phase 3 성공 + RMAT 지정으로 BLA 제출 경로가 명확하며 2026 Q4 제출 목표.

## 임상 디자인
- 디자인: Randomized, double-blind, placebo-controlled, multicenter Phase 3
- Arms: CAN-2409(아데노바이러스-HSV-TK) + 발라사이클로비르 + 방사선 vs 위약 + 방사선
- N: Phase 3 ~201명 (중·고위험 국소 전립선암)
- Phase: Phase 3 (완료, BLA 준비 중)
- Primary endpoint: 무병 생존(DFS)
- Secondary endpoint: 전립선암 특이적 DFS, PSA 재발, 전체 생존(OS)

## 타겟 질환
**중·고위험 국소 전립선암**

중·고위험 국소 전립선암(Gleason 7+, T2c-T3)은 방사선 단독 치료 후 10년 DFS 50–70%로 재발률이 높음. 국소 면역 강화 전략으로 방사선 치료 효과를 증대.

- 환자 수: 미국 신규 전립선암 연 약 250,000명, 중·고위험 국소 약 40–50%
- 병리: 전립선 국소 종양 → 방사선 치료 + CAN-2409로 종양 세포 면역원성 사멸 유도 → 전신 항종양 면역 활성화

## 기존 치료제
방사선 단독, 방사선+안드로겐 박탈 요법(ADT). CAN-2409는 방사선과의 병용으로 국소 면역원성 사멸 유도 — 기존 치료에 추가하는 애드온 전략.

## 사전 공개 임상
- **CAN-2409 Phase 3 전립선암 (방사선 병용) · NCT01436968 · Phase 3**
  결과(2025 ASCO GU): DFS HR 0.70(p=0.016), 전립선암 특이적 DFS HR 0.62(p=0.005) — 1차 endpoint 달성. 전체 생존 개선 경향 (미성숙 데이터).
- **CAN-3110 Phase 1 재발성 신경교종 · NCT03152318 · Phase 1**
  결과: 안전성 확인, OS 중앙값 개선 신호. 단독 및 pembrolizumab 병용 코호트 탐색.

## Modality
**CAN-2409 (아데노바이러스-HSV-TK)**
복제 불가 아데노바이러스 벡터 기반 자살 유전자 치료제 (type: Gene Therapy)

복제 불가(replication-incompetent) 아데노바이러스에 단순포진바이러스 티미딘 키나제(HSV-TK) 유전자 탑재. 종양 내 직접 주사 후 발라사이클로비르(경구) 투여로 HSV-TK가 전구 약물을 독성 대사물질로 전환 → 종양 세포 선택적 사멸 + 면역원성 세포 사멸.

## MOA
- 타겟: 전립선암 세포 (HSV-TK 유전자 전달 대상)
- 기전: ① 아데노바이러스가 종양 세포에 HSV-TK 유전자 전달; ② 발라사이클로비르 경구 투여 → 세포 내 HSV-TK가 간사이클로비르(GCV) 형태로 인산화 → DNA 합성 억제 → 세포 사멸; ③ 면역원성 세포 사멸(ICD) → DAMPs 방출 → 수지상세포 활성화 → 전신 항종양 T세포 반응
- 병태생리: 종양 국소 주사로 HSV-TK 발현 세포만 선택적 사멸. ICD 기전으로 국소 치료가 전신 항종양 면역을 촉발하는 "자가 백신" 효과. 방사선 병용 시 ICD 시너지.
