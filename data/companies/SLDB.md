# Solid Biosciences (SLDB)

## 카탈리스트
SGT-003 IMPACT DUCHENNE Phase 3 IND 준비 / Phase 1/2 INSPIRE 추가 데이터; FDA Type C 미팅 긍정 완료

## 회사 개요
Solid Biosciences는 뒤센 근이영양증(DMD) 유전자치료를 개발하는 희귀 근육질환 전문 기업. 선도 물질 SGT-003은 차세대 캡시드(AAV-SLB101)와 차별화 microdystrophin 구조체를 결합. INSPIRE DUCHENNE Phase 1/2에서 40명 투여 완료, 양성 중간 결과. FDA Type C 미팅에서 Phase 3 IMPACT DUCHENNE 설계 합의.

## 매출
수익 없음. 임상 단계.

## 플랫폼
- AAV-SLB101: 차세대 인테그린 수용체 타겟 캡시드 (간 지향성 감소, 심근·골격근 강화)
- Microdystrophin 구조체 (차별화 설계)

## 적응증
Rare Disease (뒤센 근이영양증, Duchenne Muscular Dystrophy)

## 파트너
없음 (독자 개발)

## 매출 구조

## 임상 디자인
- INSPIRE DUCHENNE Phase 1/2: 40명 투여 완료 (2026.3 기준)
- IMPACT DUCHENNE Phase 3 (설계 합의): 무작위, 이중맹검, 위약 대조
  - 대상: 보행 가능 DMD 환자 7세 이상 ~12세 미만
  - 1차 평가변수: Time to Rise (앙와위 기립 속도) 기저 대비 변화, 18개월
- FDA가 설계 적절성 동의

## 타겟 질환
**뒤센 근이영양증 (Duchenne Muscular Dystrophy, DMD)**

DMD 유전자 변이로 dystrophin 단백질 결핍 → 근육 세포막 취약 → 진행성 근력 소실 → 호흡근·심근 침범.

- 환자 수: 미국 약 10,000~15,000명
- 병리: Dystrophin 결핍 → 근세포막 불안정 → 운동 시 손상 반복 → 섬유화 진행

## 기존 치료제
Elevidys (Sarepta Therapeutics, FDA 가속승인). SGT-003은 차별화 캡시드·구조체로 심근/골격근 발현 개선 목표.

## 사전 공개 임상
- **SGT-003 INSPIRE DUCHENNE Phase 1/2 · 40명 투여 (2026.3 업데이트)**
  결과: Positive 중간 업데이트 - microdystrophin 발현 강건, DAPC 복원 기계적 증거, 심기능 안정화·개선, 내약성 양호.
- **FDA Type C 미팅 (2026.2) · Positive**
  Phase 3 설계 합의 완료.

## Modality
**SGT-003**
AAV-SLB101 gene therapy delivering microdystrophin for DMD (type: Gene Therapy)

AAV-SLB101(인테그린 타겟 차세대 캡시드)로 심근·골격근에 차별화 microdystrophin 유전자 전달.

## MOA
- 타겟: DMD 유전자 결손 (dystrophin 결핍)
- 기전: AAV-SLB101이 인테그린 수용체 결합 → 심근·골격근 우선 형질도입 (간 지향성 감소). Microdystrophin 발현 → DAPC(dystrophin-associated protein complex) 복원 → 근세포막 안정화.
- 병태생리: Dystrophin 없으면 근세포막이 운동 스트레스에 취약. 반복 손상 → 염증·섬유화 진행.
