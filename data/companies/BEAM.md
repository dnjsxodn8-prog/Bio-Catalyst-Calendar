# Beam Therapeutics (BEAM)

## 카탈리스트
BEAM-101 (겸상적혈구병 base editing) Phase 1/2 확장 데이터 (2026년 하반기); BEAM-302 (AATD) Phase 1/2 초기 결과

## 회사 개요
Beam Therapeutics는 염기 편집(base editing) 기술 개척자. CRISPR-Cas9 이중가닥 절단(DSB) 없이 단일 염기를 화학적으로 변환. David Liu 교수 기술 기반. BEAM-101(SCD, fetal Hb 유도), BEAM-302(AATD, Z→M allele 교정)가 리드 프로그램. Pfizer와 협업 계약.

2024년 파이프라인 집중 구조조정. 현금런웨이 2026년 말 이상 유지.

## 매출
협업 수익 소액 (Pfizer 계약금 등)

## 플랫폼
- Base Editing (adenine/cytosine base editor)
- Prime Editing
- LNP 전달 (간 타겟)
- 조혈줄기세포(HSC) ex vivo 편집

## 적응증
Hematology (SCD, β-thalassemia), Rare Disease (AATD, PCSK9 심혈관)

## 파트너
- Pfizer (다수 적응증 협업, ~$300M 계약)
- Apellis (보체 질환)

## 매출 구조
협업 수익 100% (임상 단계)

## 자체 판매
없음

## 상업화 제품
없음. 파이프라인: BEAM-101 (SCD/β-thal Phase 1/2), BEAM-302 (AATD Phase 1/2), BEAM-201 (AML CAR-T Phase 1)

## 임상 디자인
- 시험명: BEACON (BEAM-101)
- NCT: NCT05456880
- Phase: Phase 1/2
- N: ~30명 목표 (SCD 성인)
- 디자인: Open-label, dose-escalation, single-arm
- Primary endpoint: 안전성, HbF(태아 헤모글로빈) 유도율
- Secondary endpoint: VOC(혈관폐색 위기) 빈도 감소, 수혈 의존도

## 타겟 질환
**겸상적혈구병 (Sickle Cell Disease, SCD)**

- 환자 수: 미국 ~10만 명
- 병리: β-globin HBB Glu6Val 돌연변이 → 겸상 헤모글로빈(HbS) → 저산소 시 적혈구 겸상화 → VOC, 만성 빈혈, 장기 손상. HBF(γ-globin) 유도 시 HbS 중합 억제.

## 기존 치료제
Casgevy (ex vivo CRISPR, Vertex/CRISPR Tx), Lyfgenia (lentiviral gene therapy, Bluebird). BEAM-101은 base editing으로 DSB 없이 BCL11A 인핸서 편집 → HBF 유도.

## 사전 공개 임상
- **BEACON (BEAM-101) · NCT05456880 · Phase 1/2**
  결과 (2024): 초기 코호트 — 치료 환자에서 HbF 40~60% 유도 확인 (임시 데이터). VOC 0건 (추적 기간 중). 안전성 관리 가능 수준.
- **Phase 1 (BEAM-302 AATD) · NCT06038721 · Phase 1**
  결과: 용량 탐색 중. Z-allele → M-allele 교정 in vivo LNP 전달 개념 증명 완료 (전임상).

## Modality
**BEAM-101**
Ex vivo adenine base editor (ABE) targeting BCL11A erythroid enhancer in HSCs (type: Gene Therapy)

조혈줄기세포를 체외 추출 → ABE8e-SpRY 편집기로 BCL11A 인핸서 표적 A→G 변환 → BCL11A 발현 억제 → γ-globin 발현 재활성화 → HbF 증가.

## MOA
- 타겟: BCL11A erythroid enhancer (적혈구 계통 BCL11A 발현 조절 인핸서)
- 기전: ABE(Adenine Base Editor)가 BCL11A 인핸서의 GATA-1 결합 모티프 내 아데닌을 이노신(→구아닌)으로 변환 → 적혈구 계통 BCL11A 발현 억제 → γ-globin 전사 탈억제 → HbF 합성 증가 → HbS 중합 억제.
- 이중가닥 절단 없이 단일 염기 변환 → indel(삽입/결실) 최소화 → 안전성 개선 기대.
